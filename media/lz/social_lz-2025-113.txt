04 

目的地介绍 

南宁 

南宁,取南疆安宁之意,位于广西南部,千年古城,壮乡首府,中国绿城。辖7区5县,总人口约740万。气候温和,年平均气温21.7度,一年四季花常开、树常绿,绿在城中,城在绿中,适合宜居的城市。 

德天瀑布 

德天瀑布,位于崇左市大新县德天村而得名。是世界第四、亚洲第一的跨国瀑布。也是“中国最美的瀑布”和多部影视外景拍摄基地。与越南高平省重庆县山水相连,形成了中越德天跨国瀑布,近观越南板约瀑布,远观越南乡村景象。 

太平古城 

位于崇左市江洲区左江江畔,太平古城垣始建于明洪武五年(1372年),古城三面被左江环绕,占据着得天独厚的自然资源,依托着峻山秀水,美景天然而成,往古有“丽水四折,环其三面,其形若壶,故名壶城”之说。 

三娘湾 

位于钦州市钦南区犀牛脚镇南面。三娘湾是国际海豚公园是国家AAAA级旅游景区、广西十佳景区、国家一级保护动物中华白海豚之乡,也是全球唯一近距离观看中华白海豚的地方。 

北海 

地处广西南端,北部湾东海岸,背靠中国西南,面向东南亚,毗邻粤、港、澳,地处华南经济圈、西南经济圈和东盟经济圈的接合部,处于泛北部湾经济合作区域接合部的中心位置,是中国西部地区唯一列入全国首批14个进一步对外开放的沿海城市,也是中国西部唯一同时拥有深水海港、全天候机场、铁路和高速公路的城市。 

北海银滩 

位于北海市区东南面,距市中心约10公里。北海银滩原名叫“白虎头沙滩”“龙虎银滩”,北海市政府在建设旅游区时最后选定了“北海银滩”这个名字。“北海”两个字点明了位置,“银滩”两个字也显文雅。 

鳄鱼山火山地质公园 

是涠洲岛最核心的景区之一,经过上百次的火山喷发而形成了现在岛上的地层主体。后又多次发生海洋风暴、地震、海啸,加上平时风浪的作用,形成了现在涠洲岛丰富多彩的海蚀、海积与海滩地貌。 

石螺口 

是涠洲岛观日落最美的地方,而且每天的日落都不一样,吸引了众多摄影爱好者前来打卡。一到傍晚海边全是长枪短炮,那阵势也算是一道亮丽的风景线吧。 

涠洲岛 

位于北海市的北部海面上,南望海南岛,北靠北海市,东邻雷州半岛,西近越南;距北海市区24海里,是中国十大魅力海岛之一,也是广西最大的海岛。岛上75%以上是客家人,这里冬无严寒、夏无酷暑,风光旖旎,被称为是“坠落在大海上的天堂”。涠洲岛盛产香蕉、菠萝蜜、芒果、海菠萝等水果。 

五彩滩 

因巨大的火山岩石一层一层,在阳光的照射下特别壮观;大片大片的火山熔岩滩裸露出来,但在上面仍有一洼洼的积水,在蓝天的映射下,这些积水也变成蓝色,和裸露的岩石一起形成了五彩斑斓的景色。五彩滩也是涠洲岛观日出最美的地方。

详细行程 

D1:全国各地-南宁(安排接站/接机) 

 宿:南宁 

下午或晚上抵达南宁站或机场后,专人接车或接机,前往酒店入住。晚上自行前往南宁市西关路小吃一条街或南宁新地标【三街两巷】 

品尝当地特色美食,后观邕江夜景或前往南宁夜市新地标【南宁之夜】观夜景及品尝美食。 

D2:南宁-中越德天跨国大瀑布-崇左太平古城(含早餐) 

 宿:崇左 

早餐后,乘车前往硕龙镇德天瀑布游客中心(230KM/3.5H),抵达后,换乘景区观光车,前往德天村(10KM/20分钟),游览中越两国界河归春河,领略具有异国情调神秘奇特的边关民族风情,到达宽200多米,落差70多米,三级跌落的中国最美的瀑布—【德天跨国大瀑布】(车程约1小时, 游览约3小时) 

感受亚洲第一大跨国瀑布的雄伟和神奇,远眺和德天瀑布相连的越南板约瀑布,中越53号界碑,可登高望越,远观越南乡村景观。下午乘车前往广西国门城市-崇左(车程约3.5小时)抵达后入住酒店,晚上可前往崇左新地标、世界壮乡第一古城-【太平古城】 

观夜景及寻找美食。崇左市太平古城历史悠久,宋太宗太平兴国年间(976-984 年),北宋设置左江道太平寨,治所设在今崇左市江州区太平街道壶兴社区,太平之名由此而来,其后元、明、清各朝先后设太平路和太平府,至今已有1000多年历史。整个景区围绕骆越文化、壮乡文化、水文化、糖文化等主题,在原址上复原并扩建了内河,修缮了古城墙。是集城市配套、民俗体验、风情美食、休闲娱乐、文创区和主题民宿、当地特色民俗文化产品展销、民俗风情体验等多种业态于一体,能够满足旅客“游、购、玩、吃、住、行”一站式消费体验的4A级文化旅游景区,致力于打造成崇左首席大型文旅综合体,营造有别于城市商业中心全新消费体验,推动商业开发与民俗演出、科技管理、非遗传承等融合发展。 

D3:崇左-钦州三娘湾国际海豚公园-北海银滩-老街(含早餐) 

 宿:北海 

早餐后,乘车前往钦州犀牛脚镇,游览【三娘湾国际海海豚公园】(车程约2小时) 

三娘湾以碧海、沙滩、奇石、绿林、渔村、海潮、椰林、中华白海豚著称。 

是著名电影《海霞》、《龙之战》、36集电视剧《沧海丝路》和中央电视台MTV《湾湾歌》外景拍摄地。参观有美丽动人传说的三娘石,石猪赶潮、石狗镇海、天涯石等景点。在沙滩上、浅海中,各种奇石林立,千姿百态,惟妙惟肖、令游人叹为观止。抵达后,【乘船出海】(出海时间约1小时),观中华白海豚,成群结队的在大海中嬉戏也可撒网捕鱼。中餐后,乘车前往国家4A级景区【北海银滩】(车程约2小时), 

北海银滩是中国唯一建在少数民族自治区的国家级旅游度假区。度假区位于北海市南部海滨,距北海市区8公里,由西区、东区和海域沙滩区组成。银滩海滩延绵约24公里,海滩宽度在30-3000米之间,总面积约38平方公里,均由高品质的石英砂堆积而成,在阳光的照射下,洁白、细腻的沙滩会泛出银光,故称银滩,北海银滩到其"滩长平、沙细白、水温净、浪柔软、无鲨鱼、无污染"的特点,被喻为"天下第一滩"。行程结束后,乘车前往酒店入住,晚上乘车前往【百年老街】 

沿街全是中西合璧骑楼式建筑。这些建筑临街的骑楼部分,既是道路向两侧的扩展,骑楼的方形柱子粗重厚大,颇有古罗马建筑的风格,老街从入口开始往东走,有着舌尖上中国美食之称的美食店有:千古厘岛,老道咖啡,李姨虾饼店,宽记手工冰淇淋,细辉螺吧,庞记炒螺,宽记甜品等。

这儿是一座在市区看不到海的海滨城市,却也因此,拥有野性原始的海滨风光。长达562公里的海岸线绵延,虽不像三亚那般清澈湛蓝,却没多少开发痕迹,反而多了一种野性,深沉和苍茫。

都说海南的海滨公路是绝色,却不知这人也有一条公路。长达13公里的最美海上公路,直插入海,仿佛在海上奔腾。难怪大家要说有海的城市,总会浪漫一些。

/ 三娘湾 / 三娘湾被誉为“中华白海豚之乡”,是钦州最最浪漫的地方。

大海、蓝天、阳光、沙滩、渔船、海豚彼此不期而遇,在这方天地间汇聚成绝美动人的风景线。

等一个晴好的天气带着相机悄悄出海,一望无际的碧色海水中盛满了期待,说不准就能碰到难遇的粉色海豚。

坐在并不太稳当的小船上,吹着湿咸的海风,突然望见波光潋滟的海面上连续跳跃着几只可爱生灵,唯美梦幻又惊喜十足。

除了看海豚,在这儿守候一场日落也是一件极为浪漫的事儿。夕阳在云层间卷着金光,风吹得海面稍显凌乱,远处灯塔悄悄驻立,渔船静静停泊,海鸥划破淡橘色天际。

夕阳西下时,这里的海湾柔和安静,好像是一方没有压力和996的净土。海风一吹,烦恼也渐渐消失在云霞里,只剩下了广西最动人的诗意。

/ 三墩港 / 三墩港被誉为钦州的“小垦丁”,旅行攻略查无此人,但是在小红书上爆火的海港。

三墩的范围很大,既是一座港口,也包括了一座人造的小小海岛。沿着海上公路一直开就可以看到绝美的景色。

绵延13公里的海上公路贯穿大海中央,电影感十足。公路两旁栽满了树木,远远望去像是邂逅了海上的绿洲,美妙至极。

小岛上面还有牛羊,还有昏黄的草地,文艺气息瞬间拉满。这里也很适合来一场海边的野餐,对着大海吹吹风聊聊天。

/ 犀丽湾 / 犀丽湾又称月亮湾,是钦南地区私藏的小众海域。一湾海水,一片椰林,一条通向海角的路。

漫步犀丽湾,优哉游哉地饱览这海天一色的绝美景致,不受打扰,难得清闲。

环顾四周,吊床、木船、茅草伞、秋千、躺椅等一应俱全,它们零星散落在干净的沙滩上,一秒便能将度假氛围拉满。

/ 茅尾海 / 茅尾海其实是一片海。它有着首批国家级海洋公园,同时也悄悄的顶着“中国最美内海”的名头。

总面积为135平方公里的茅尾海,走在海边微风四起,旖旎的水径推动着微波层层叠叠抵达海岸,令人心神荡漾。

不止是海,这里还有万亩红树林,绿意流动,白鹭成群,美不胜收,可以来一场和阳光邂逅的下午茶。

它不仅好看,而且很好吃。处于咸淡水交汇处的茅尾海,特别适宜大蚝的生长。放眼望去,蚝排养殖遍布十里的场面极为壮观,光是看看就让人无法自持,口水四溢!

/ 乌雷村 / 而在犀牛脚镇,藏着一个面朝大海的小众村落——乌雷村,它是许多本地人都不知道的存在,270°的开阔海景让人流连忘返。

涨潮的时候,海水会淹没礁石和沙滩,近处是阵阵浪花翻涌,远处是遥遥耸立的红白灯塔,有些像是文艺电影里的场景。

这里的海风落日,也让人沉沦,你可以坐在灯塔旁,看着红日在正前方缓缓垂落,明艳的霞光在海天间肆意涂抹,渐渐消散,远处油轮码头的灯火一点点亮起。

广西的山水真的太灵了钦州,也是如此钦州,除了大海,更让瞧妹心动的是那山水相依的密林绿意。山水虽然没有桂林那样钟灵毓秀,却独具了自己的一份俏丽柔情,而且人少又安静。

/ 八寨沟 / 人人都向往四川的人间仙境九寨沟,殊不知在钦州腹地有个“清凉国”八寨沟,景致毫不逊色却少有人至。

它地处广西著名的“十万大山”北麓,传说这里原来是八个村寨,因此得名。后来随着公路的建设,这里的风光也渐渐传到了山外,成为了钦州人私藏的小秘境。

有满目绿意的原始森林,有层叠错落的瀑布,有静谧沁润的悠长曲径,有古旧沧桑的石桥吊桥……仿佛一下子闯入了“绿野仙踪”里。

这里也很适合露营!在一片葱郁中扎个帐篷,在冰凉的溪水中丢个西瓜,晚上在诗酒往事中沉沉睡去。

/ 六峰山 / 《隐秘的角落》的取景地——湛江。但实际上,剧中开头很重要的一部分——六峰山,其实位于广西的钦州。

钦州六峰山自古便有“人寰胜地”的美誉。这里是典型的喀斯特地貌,由龙头、凤尾、鹤立、龟背、宝障、冲霄六座主峰组成。

一路沿台阶攀行,目之所及尽是参天古木,怪石嶙峋,云雾缭绕,风光旖旎,美到饱和,活脱脱是仙境本境。

/ 五皇山 / 在钦州的东部,还藏着一座茫茫的高山草甸,有着不输武功山的自在与清凉。

坐着缆车登上了半山腰,大片的苍绿,起伏的山坡,耸立的怪石,洁白的风车,一下子跃入了眼帘,那么壮观,那么惊艳!

你可以沿着步道肆意奔跑,两旁尽是摇曳的山花,繁茂的林木涌起了绿浪,山间的风车悠悠地转动,低垂的白云仿佛伸手可及……

水果自由!生蚝自由!嗦粉自由!钦州真的太好吃了在整个广西地区,随便一个城市单独揪出来都是活的美食地图。当然,钦州也不例外。这里水果自由、海鲜自由、嗦粉自由、糖水自由、小吃自由,也太哇塞了吧!

/ 嗦粉 / 钦州真是个嗦粉天堂啊。猪脚粉、煲仔粉、叉烧粉、锅烧粉、牛腩粉、生榨米粉、猪杂粉……一口浓汤一口粉,实不相瞒,钦州人觅食的快乐就是这么简单。

/ 猪脚粉 / 在钦州流传着这样一句话:“吃了猪脚粉,神仙也打滚”。

一碗连汤带水、热气腾腾的猪脚粉被端上来,慰借着城市早起苏醒的灵魂。

韧滑、清爽的卷粉浇上一勺醇香浓厚的猪脚汤,放上几块猪脚,撒点葱花,加点开胃的酸菜,就是无数钦州人从小吃到大的味道。

纯纯淡淡的猪油香气沥在汤里,猪脚浓香,粉也烫的刚刚好。大口大口嗦粉,汤鲜味美,爽滑又过瘾,吃完感觉嘴角都有留香。

/ 叉烧粉 / 来钦州一定要吃碗叉烧粉。满满一碗叉烧肉堆在柔韧的粉上,冒着热气分量十足。

叉烧肉偏瘦富有嚼劲,有股炭烤的香味,最精髓的绝对是汤底,看似平凡却非常抓人胃口。

清晨骑着电车停靠在一家招牌陈旧的老字号小店前,嗦一碗叉烧粉,望着无处不在的人间烟火,便能充满一整天的元气。

/ 煲仔粉 / 广东煲仔饭很有名,而钦州的煲仔粉也不输,一碗煲仔粉,材料是肉眼可见的丰富。

虾、猪杂、鹌鹑蛋、螺肉、牛肉、鱿鱼、豆芽、青菜等满满当当地铺在入味的粉上。

巨鲜巨香,要搭配着灵魂的辣椒酱,一口销魂,太馋人了!

/ 生蚝 / 海鲜爱好者必然知道钦州是有名的“大蚝之乡”,全国约60%的蚝苗都出自于此。

在“养蚝大户”钦州,嘬一口又大又鲜的生蚝最是过瘾,添着冰块的奶白色大蚝端上来的那一瞬间,口水就难以抑制了。

除了冰镇生蚝,烤生蚝也同样拉满了瞧妹的期待。

丰满肥嫩的蚝肉在炭火炉上滋滋作响,撒些蒜蓉淋上秘制酱汁,再以葱花点缀,香气爆炸式四处散发,鲜到上头根本停不下来。

/ 水果王国 / 处于湿热多余的亚热带气候下的钦州,盛产荔枝、火龙果、芒果、龙眼、芭乐、菠萝蜜、沃柑等,种类繁多,轻松就能实现水果自由。

去一趟钦州农贸市场,各种水果琳琅满目让人看花了眼不说,重点是价格也低得让人惊掉下巴!

多汁的甘蔗、绵密的榴莲、醇香的芒果、可口的木瓜以及甜到心尖的荔枝,又是被水果俘获的一天,爱了爱了。

/ 糖水 / “夏饮绿豆沙,冬食芝麻糊”,钦州人也偏爱各种各样的清甜糖水。在老城的街巷里,市场间,总能寻到卖糖水的小铺子。

在酷暑难耐的午后,去买上一碗冰冰凉凉的糖水,或是清爽解腻的绿豆汤,或是入口滑嫩的槐花粉,或是细嫩绵软的豆腐花……丝丝凉意在口中化开,刹那荡去燥热。

有些糖水铺还会采取自助式服务,花上4、5块,各色的糖水小料随便加,缤纷的果块,绵软的芋圆,带些嚼劲的红豆,炸得酥脆的蛋散……瞬间掀起甜品的狂欢!

/ 蛋散 / 要是让钦州人选出当地的传统小吃担当,那么灵魂蛋散绝对是当仁不让。

香脆浓郁的蛋散,一直以来都是钦州人用来配糖水吃的不二选择。

咬一口嘎嘣脆的蛋散,酥酥脆脆中竟体味到一丝丝香甜,此时饮一大口甜蜜的糖水,不知牵动的是多少钦州人的童年回忆。

/ 水糕 / 每一个来过钦州的人,都会爱上白白胖胖的水糕的味道。

一层层的米浆蒸熟后口感嫩滑,再铺上白萝卜、碎肉、萝卜丁和木耳料,满满一层淋上豉油,吃起来层次感丰富,整体口感实在是妙。

/ 卷粉 / 一条通体晶莹的卷粉,是无数钦州人早起的唤醒方式。

在蒸熟的粉皮中放入炒好的木耳、豆角等然后卷好,薄薄的一层隐约间都能看见馅料。接着浇上微甜的黄皮酱,撒点炸虾皮,卷粉就做好了。

一口咬下这弹爽嫩滑的美味,一时间米粉的香气、馅料的丰富与酱汁的惊喜在口中化开,吃到就是赚到~

           钦州的山水,美丽又舒适 

         1 

       八寨沟 

众人皆知四川有个美丽的九寨沟,殊不知在钦州,还有一个并不逊色的八寨沟。它地处十万山北麓,属于4A级景区。 

传说这里原来是八个村寨,村民们在这八公里长的山沟里以采摘八角为生,因此得名。 

这里仿佛是一片处女地,满眼都是亚热带雨林、清澈的山涧和瀑布,还有大小各异的泉潭和奇特的砂页岩地貌。而且这里的负氧离子含量极高,是很多年轻人露营和野餐的好去处 

          2 

        六峰山 

典型喀斯特地貌的六峰山,由龙头、凤尾、鹤立、龟背、宝障、冲霄六个山峰组成。今年爆火的网剧《隐秘的角落》“爬山”画面同款,就是在这里取景! 

山中岩洞大小共有60多个,这里有石碑、庙宇,还有八百多年前的崖刻和摩崖造像,历史和艺术价值也是极高的。 

景区内自然景观和人文景观相得益,傍晚的时候站在山顶俯瞰这座小城,也是别样的舒适。 

          3 

        五皇山 

距离县城30多公里,五皇山旅游风景区位于中国蕉乡、世界长寿之乡—广西钦州浦北县境内,是全国最大的红椎林保护区。 

景区内具有“山高林茂、草绿石奇和穀幽水秀”的特点,是一个大型天然氧吧。 

      图源@钦州旅游 

乘坐高空缆车俯瞰山下的景观,云海苍茫会让人有种在仙境般梦幻感,到了晚上,繁星闪烁,同样是一片美好景象。 

           钦州的老街,古老有韵味 

          1 

        中山路 

中山路是钦州保存的最好的一条老街,在这里,可以让你重新认识这个从民国时期就兴起的沿海港口。 

      图源@Fannie Li 

这里至今也没有经历过大规模的翻修,路上的商铺以售卖古玩、古手艺为主,还有各种具有岭南风情的骑楼,蕴藏着钦州最古老的人文景观。 

图源@Fannie L 

推荐四五点的时候来这里,先逛逛老街,穿梭在老街的巷子里看夕阳。或是进行一次逛吃逛吃的旅程,一定会让你回味无穷! 

图源 @欧米米伽 

          2 

        灵山大芦村 

灵山大芦村是广西三大著名古村之一,已有五百余年的历史。 

大芦村又有“荔枝村”之称,村里处处可见几百年树龄的荔枝树、龙眼树和香樟,还有着广西保护最完好、规模最大的明清建筑群,同时这里也是灵山有名的摄影拍摄基地。 

漫步在古村街头,既像是行走在一幅风景优美的水墨画中,同时又能感受到村落浓郁的烟火气息。 

          3 

        钦州古龙窑 

钦江古龙窑不仅是国内最长的古龙窑,也是传承千年坭兴陶的龙图腾,极具文物保护和利用价值。 

600多年期间数次更换窑址,历经了九座古龙窑的窑火传承,见证和代言了中国坭兴陶的辉煌和起伏。 

           钦州的海,浪漫又梦幻 

          1 

        茅尾海 

茅尾海是个富饶美丽的半封闭内海,它是首批国家级海洋公园,同时也有“中国最美内海”的美誉。

面积135平方公里,旖旎的水径和秀丽的小岛交融在一起,风光无限好。 

咸淡水交汇的茅尾海还是中国南方最大的天然蚝苗采苗和人工养殖基地,这里长出来的蚝,个大、肉肥、味美。在这里,也可以打卡在《舌尖上的中国》才看到的养蚝景象,一饱眼福。 

          2 

        三娘湾 

三娘湾被誉为"中华白海豚之乡”,是我们国家少数几个能在近海看见珍贵白海豚的海域。 

这里有着碧海和沙滩、千姿百态的礁石和清丽恬静的绿林等大自然景观,同时也有着古朴温馨的渔村文化。 

在这里,除了可以乘船出海观看珍稀的海豚,还可以朝起观旭日东升,晚赏夕阳斜照,感受渔村的宁静和美好,一样怡然自得。 

          3 

        三墩 

三墩有海,有戈壁,还有最美的海上日落,美丽的景色让它拥有了“广西版小垦丁”的称号。 

      图源@LLY 

四面环海,长达13km的填海公路,开一辆车,就可以把360°无死角的美景尽收眼底,邂逅浪漫的海上风景。 

公路的尽头有一个小沙滩,在海边静静地观看日落,感受时间的流逝,十分惬意。而且这里的海鲜便宜又好吃,记得好好吃一顿! 

          4 

        白沙湾 

相对于前几个沙滩,位于江山半岛区域的白沙湾是个小众、尚未开发完全的海滩,人少但景美。 

图源@ 这里有个悬疑妹 

海边上的礁石和灯塔充满着日系的感觉,随手都可以拍出好看的大片。沙滩西侧是灯价岭,站在岭上看夕阳缓缓下降,美不胜收。 

图源@Hanna

目的地介绍

南宁

南宁,取南疆安宁之意,位于广西南部,千年古城,壮乡首府,中国绿城。辖7区5县,总人口约740万。气候温和,年平均气温21.7度,一年四季花常开、树常绿,绿在城中,城在绿中,适合宜居的城市。

德天瀑布

德天瀑布,位于崇左市大新县德天村而得名。是世界第四、亚洲第一的跨国瀑布。也是“中国最美的瀑布”和多部影视外景拍摄基地。与越南高平省重庆县山水相连,形成了中越德天跨国瀑布,近观越南板约瀑布,远观越南乡村景象。

太平古城

位于崇左市江洲区左江江畔,太平古城垣始建于明洪武五年(1372年),古城三面被左江环绕,占据着得天独厚的自然资源,依托着峻山秀水,美景天然而成,往古有“丽水四折,环其三面,其形若壶,故名壶城”之说。

三娘湾

位于钦州市钦南区犀牛脚镇南面。三娘湾是国际海豚公园是国家AAAA级旅游景区、广西十佳景区、国家一级保护动物中华白海豚之乡,也是全球唯一近距离观看中华白海豚的地方。

北海

地处广西南端,北部湾东海岸,背靠中国西南,面向东南亚,毗邻粤、港、澳,地处华南经济圈、西南经济圈和东盟经济圈的接合部,处于泛北部湾经济合作区域接合部的中心位置,是中国西部地区唯一列入全国首批14个进一步对外开放的沿海城市,也是中国西部唯一同时拥有深水海港、全天候机场、铁路和高速公路的城市。

北海银滩

位于北海市区东南面,距市中心约10公里。北海银滩原名叫“白虎头沙滩”“龙虎银滩”,北海市政府在建设旅游区时最后选定了“北海银滩”这个名字。“北海”两个字点明了位置,“银滩”两个字也显文雅。

鳄鱼山火山地质公园

是涠洲岛最核心的景区之一,经过上百次的火山喷发而形成了现在岛上的地层主体。后又多次发生海洋风暴、地震、海啸,加上平时风浪的作用,形成了现在涠洲岛丰富多彩的海蚀、海积与海滩地貌。

石螺口

是涠洲岛观日落最美的地方,而且每天的日落都不一样,吸引了众多摄影爱好者前来打卡。一到傍晚海边全是长枪短炮,那阵势也算是一道亮丽的风景线吧。

涠洲岛

位于北海市的北部海面上,南望海南岛,北靠北海市,东邻雷州半岛,西近越南;距北海市区24海里,是中国十大魅力海岛之一,也是广西最大的海岛。岛上75%以上是客家人,这里冬无严寒、夏无酷暑,风光旖旎,被称为是“坠落在大海上的天堂”。涠洲岛盛产香蕉、菠萝蜜、芒果、海菠萝等水果。

五彩滩

因巨大的火山岩石一层一层,在阳光的照射下特别壮观;大片大片的火山熔岩滩裸露出来,但在上面仍有一洼洼的积水,在蓝天的映射下,这些积水也变成蓝色,和裸露的岩石一起形成了五彩斑斓的景色。五彩滩也是涠洲岛观日出最美的地方。

D3:崇左-钦州三娘湾国际海豚公园-北海银滩-老街(含早餐) 宿:北海早餐后,乘车前往钦州犀牛脚镇,游览【三娘湾国际海海豚公园】(车程约2小时)三娘湾以碧海、沙滩、奇石、绿林、渔村、海潮、椰林、中华白海豚著称。是著名电影《海霞》、《龙之战》、36集电视剧《沧海丝路》和中央电视台MTV《湾湾歌》外景拍摄地。参观有美丽动人传说的三娘石,石猪赶潮、石狗镇海、天涯石等景点。在沙滩上、浅海中,各种奇石林立,千姿百态,惟妙惟肖、令游人叹为观止。抵达后,【乘船出海】(出海时间约1小时),观中华白海豚,成群结队的在大海中嬉戏也可撒网捕鱼。中餐后,乘车前往国家4A级景区【北海银滩】(车程约2小时),北海银滩是中国唯一建在少数民族自治区的国家级旅游度假区。度假区位于北海市南部海滨,距北海市区8公里,由西区、东区和海域沙滩区组成。银滩海滩延绵约24公里,海滩宽度在30-3000米之间,总面积约38平方公里,均由高品质的石英砂堆积而成,在阳光的照射下,洁白、细腻的沙滩会泛出银光,故称银滩,北海银滩到其"滩长平、沙细白、水温净、浪柔软、无鲨鱼、无污染"的特点,被喻为"天下第一滩"。行程结束后,乘车前往酒店入住,晚上乘车前往【百年老街】沿街全是中西合璧骑楼式建筑。这些建筑临街的骑楼部分,既是道路向两侧的扩展,骑楼的方形柱子粗重厚大,颇有古罗马建筑的风格,老街从入口开始往东走,有着舌尖上中国美食之称的美食店有:千古厘岛,老道咖啡,李姨虾饼店,宽记手工冰淇淋,细辉螺吧,庞记炒螺,宽记甜品等。傍晚时分

遇龙河漂流这一特色旅游项目,让人们能够在山水之间尽情享受漂流的乐趣,感受与大自然亲密接触的美妙。

湛江位于角尾乡,地形似牛角,有壮观的双海奇观,琼州海峡和北部湾的海水在此交汇,还有珊瑚礁国家级自然保护区,拥有着极其美丽的自然景观,让人应接不暇。    南极村      通常所说的“南极村”指的是中国大陆南极村,位于广东省湛江市徐闻县西南部的角尾乡。    “北纬20°13′14″,代表南极村的浪漫,一生一世的约定”。   南极村还可以看见许多壮丽的景观,使人们叹为观止。灯楼角矗立着一新一旧、一高一矮的两座灯塔,旧灯塔建于1884年,由法国殖民政府建造,新灯塔于1994年重建,高36米,是琼州海峡和北部湾唯一的航标灯塔,也是中国大陆最南端的标志性建筑。徐闻古港是海上丝绸之路的始发港,位于南山镇大汉三墩旅游区东北400米,港内有八角形的古航标灯座,见证了曾经的繁华。不经如此,徐闻还有着丰富的文化内涵与名俗文化。如石狗文化,雷州歌对唱、舞龙舞狮、游神等,其中雷州歌对唱是当地人民表达情感、传承文化的重要方式。菠萝的海位于曲界镇,数十万亩菠萝连片成海,被誉为“菠萝的海”,是中国最大的菠萝生产基地。赤坎老街早在宋朝,赤坎作为海畔商埠就已成雏形,原为盐埠,明清海禁时港埠几近废弃,康熙收复台湾废止海禁后再度兴旺,清道光年间成为粤西商业物流中心。这里有十九世纪末、二十世纪初欧式风格建筑,如广州湾商会会馆,模仿法国风格设计的钟楼高耸。还有岭南骑楼建筑,如赤坎民国风情街的骑楼,廊道相连,遮阳避雨,适合南方气候。

阳江海陵岛位于中国广东省阳江市西南部,是广东第四大海岛,也是国家AAAAA级景区。海陵岛距阳江市区20千米,该岛东西长约23.5千米,南北宽仅3~10.5千米,其主岛面积108.89平方千米,海域面积640平方千米,主岛海岸线长75.5千米,浅海和滩涂面积98平方千米,是广东第四大海岛,是阳江市最大的海岛。(1)特色景观:大角湾,十里银滩,马尾岛,北洛湾等景观(2)人文景观:海上丝绸之路博物馆,闸坡风景区,南海放生岛我想大家都看过《狂飙》吧,而广东也是其取景地,接下来,我们将一起走进《狂飙》取景地--梅家大院         梅家大院(汀江圩华侨建筑群),位于广东省台山市端芬镇大同河畔,是一处融合了历史与现代、东方与西方的建筑瑰宝。始建于1931年,曾是商贾云集的繁华之地,是当时台山乃至整个五邑地区的商贸中心       大院占地面积80亩,108幢二至三层带骑楼的楼房呈长方形排列,鳞次栉比,整齐划一,中间有40亩专供商贩摆卖商品的市场空地,俨如一座小方城。建筑物既表现出欧美国家的建筑风格,又体现了中国传统的建筑艺术,每幢楼宇规划整齐,但外型各异。

珠海    珠海是一座极具魅力与特色的城市,:濒临南海,东与香港水路相距36海里,南与澳门陆地相连,是内地唯一与香港、澳门同时陆路相连的城市。是中国重要的口岸城市,设有10个国家一类口岸,是仅次于深圳的中国第二大口岸城市。情侣路  是珠海的城市名片,沿途可欣赏珠海渔女雕像等景观;珠海情侣路是广东省珠海市的沿海市政道路,分情侣南路、情侣中路、情侣北路,全长28公里,也是珠海的浪漫名片

珠海日月贝即珠海大剧院,是中国第一座海岛大剧院,也是珠海市的地标性建筑。     由陈可石设计,以珠海盛产的日月贝为构思主题,以名画《维纳斯的诞生》为灵感,整体造型为一大一小的日月双贝,“大贝壳”高90米,“小贝壳”高60米,白天半通透,夜晚灯光闪耀。      蓝色的天空和海洋与白色的贝壳型建筑相互映衬,蓝白组合浪漫且充满诗意,周边景观设计以鱼类的鳍为灵感,强化了滨海建筑特色。港珠澳大桥是连接中国香港、广东珠海和中国澳门的跨海大桥,是世界上最长的跨海大桥,全长约55公里。由桥梁、隧道和人工岛组成,主体工程由6.7公里的海底沉管隧道、22.9公里的桥梁、逾20万平方米的东西人工岛组成,即“桥—岛—隧”一体。拱北口岸是连接中国澳门特别行政区与广东省珠海市的陆路口岸。促进商业经济贸易,创造就业机会,推动珠海发展孙中山故居通常所说的孙中山故居指的是孙中山故居纪念馆,位于广东省中山市南朗镇翠亨大道93号。       1866年孙中山诞生于此,1892年他兄长孙眉汇款回乡,由孙中山主持建成了一栋中西合璧的两层楼房。1956年孙中山故居纪念馆成立,1999年重建后的纪念馆落成并向观众开放。       故居为中西合璧的建筑风格,融合了西方建筑的元素和中国传统建筑的特色,外观独特,内部布局合理,是中国近代建筑的杰出代表之一。

广州小蛮腰广州塔广州塔  塔身主体高454米,天线桅杆高146米,总高度600米,是国家AAAA级旅游景区,总建筑面积102000平方米。

     拥有多个观光平台,如488米户外观景平台,曾获吉尼斯世界纪录“最高观景平台”称号;450米户外观景平台则是较为实惠的欣赏塔顶风光的户外平台。

北京路步行街位于广东省广州市越秀区中部偏东南,是广州建城2200多年来未偏移、未断代的城市中轴线。       秦始皇三十三年,南海郡尉任嚣在北京路一带筑番禺城作为南海郡的郡治,是广州建城之始。北宋拆南城墙,建“双阙”,后改为双门城楼,北京路中段被称作“双门底”,渐渐成为商业区。2016年8月,北京路文化旅游区正式挂牌国家AAAA级旅游景区。           来到广州怎么能不夜游珠江呢?    夜幕降临,两岸建筑的灯光亮起,勾勒出建筑的轮廓,在江水的映照下,形成五彩斑斓的光影,如广州塔的灯光变幻,将小蛮腰衬托得婀娜多姿。       解放大桥、海珠桥等桥梁各具特色,灯光映照下,犹如横跨江面的璀璨彩虹。      游船驶过,江面波光粼粼,江水在灯光与月光的交织下,闪耀着细碎的银光,灵动而迷人,夜晚的江风轻柔凉爽,吹走了白天的疲惫与燥热,让人顿感神清气爽,心情也随之放松。

东莞      东莞的林则徐纪念馆,又称虎门林则徐纪念馆,是为纪念清朝民族英雄林则徐而建。1958年1月1日正式对外开放,其前身为1957年成立的林则徐公园和林则徐纪念馆建设筹备小组所筹建的林则徐纪念馆。      该纪念馆由虎门林则徐纪念馆、海战博物馆、沙角炮台管理所和威远炮台管理所组成,管理面积约80万平方米。     同时记录着民族英雄林则徐的辉煌一生!

中英街

       中英街是位于深圳市盐田区沙头角街道与香港特别行政区北区交界处的一条著名街道。      长约250米,宽约3—4米,街心以“界碑石”为界,深圳和香港各占一半。街边商店林立,有来自五大洲的产品,品种十分齐全。有步行街、环城路、海傍街、横头街、碧海路等5条街道与道路,主要建筑有入口广场、骑楼、回归广场、天后宫广场等。      是深圳特有的“一街两制”街区,融合了深港两地的文化特色,有独特的历史景观和文化风情。自2002年开始,每年3月18日举办“中英街3.18警示日”鸣钟活动,是爱国主义教育品牌活动。2012年6月8日,被评为中国历史文化名街。

世界之窗      深圳世界之窗是国家AAAAA级旅游景区。  世界广场:位于入口处附近,有环球舞台、世界文化浮雕墙、中华门、埃及门等建筑   欧洲区:位于北部,有法国埃菲尔铁塔(仿)、希腊雅典卫城(仿)、法国巴黎凯旋门门(仿)等景点。    亚洲区:位于西南部,有印度尼西亚婆罗浮屠(仿)、印度泰姬·玛哈尔陵(仿)、泰国泰王宫(仿)等景点。   大洋洲区:位于西北部,有澳大利亚悉尼大铁桥(仿)、悉尼歌剧院(仿)等景点。    宋井:相传为南宋末代皇帝赵昺南逃时所挖,虽身处海滩,却常年不被海水淹没,水质甘甜,是南澳岛的一大奇观。金银岛南澳海岛国家森林公园汕头    “潮起汕头,韵动古今”:“潮起”寓意着汕头作为沿海城市,得改革开放风气之先,发展浪潮奔涌向前;“韵动古今”则体现了汕头既有深厚的历史文化底蕴,如古老的潮汕文化、传统的建筑和手工艺等,又充满现代活力,像繁华的都市景象、活跃的经济发展等,古今交融,富有韵味与动感。   南澳岛-北回归线打卡点     南澳岛是广东省汕头市南澳县下辖岛屿,也是广东省唯一的海岛县,位于广东省东南部海面,北临柘林湾,南临南海,处于闽、粤、台三省交界海面,距西太平洋国际主航线仅13海里,素有“粤东屏障、闽粤咽喉”之称。     青澳湾被称为“东方夏威夷”海湾呈弯月形,南北走向,水深5-10米,泥沙底,有1条小溪注入,东北面是天然避风港,湾内栖息着中华白海豚、绿海龟等国家一级保护野生动物。汕头小公园指的是广东省汕头市金平区内的一片老城区,又被称为“小公园开埠区”。以中山纪念亭为核心,向周边呈扇形放射状分布着安平路、升平路、国平路等街道,直达梅溪河和韩江,整体布局呈现“大”字状,是当时世界少有、国内独一无二的街区。广济桥      广济桥,俗称湘子桥,位于广东省潮州市古城东门外,横跨韩江,连接东西两岸,为古代广东通向闽浙交通要津,是中国四大古桥之一。  承载着潮州的历史和文化,桥上的亭台楼阁、匾额楹联等都是珍贵的文化遗产,见证了潮州的繁荣发展,是研究古代桥梁建筑和地方历史文化的重要实物资料。      作为国家4A级景区和潮州八景之一,其独特的建筑风格和开合的桥梁形式吸引了大量游客前来参观,成为潮州旅游的标志性景点。

2024年珠海海洋工作年终成绩单之二: 

海洋十大“关键词” 

01 

海洋新质生产力

近年来,珠海不断加大对海洋创新投入和支持力度,成立了一批以南方海洋科学与工程广东省实验室(珠海)为代表的涉海科研平台,推动海洋领域由资源要素驱动转向科技创新驱动,国家海洋综合试验场(珠海)揭牌、“海基二号”完工、“香洲云”交付、海上风电制氢、海上二氧化碳回收利用项目……海洋新质生产力迎来更加广阔的发展前景,海洋科技创新得到更为广泛的应用。 

2024年10月,国家海洋综合试验场(珠海)正式揭牌并啔动首次试验,打通科技成果转化的“最后一公里”。

2024年3月,由我国自主设计建造的亚洲第一深水导管架“海基二号”在珠海深水装备制造基地建造完工,该项目刷新了结构高度、重量、作业水深、建造速度等多项亚洲纪录,入选“2024年度央企十大国之重器”。 

2024年5月,测试工作保障船“香洲云”在珠海九洲码头顺利交付。“香洲云”填补了我国在区域级科学调查船方面的空白,达到了目前国际同级别科考船的先进水平。 

2024年10月,华南地区首个海上二氧化碳回收利用项目在珠海投产。该项目是中国海油首个依托生产设施建设的二氧化碳回收利用项目,既可减少碳排放,又可提高经济效益。 

2024年12月,全球首台套兆瓦级电解海水制氢装置在珠海经开区试运行成功,开创了海上可再生能源利用新路径。 

02 

“百千万工程”现代化特色海岛 

“零碳岛”、“陆岛同价”、渔旅综合平台……2024年,珠海多措并举,助力打造“百千万工程”现代化特色海岛,持续推进“百县千镇万村高质量发展工程”,积极促进我市城乡区域协调发展。 

2024年6月,粤港澳首家离岛国际连锁高端度假酒店——格力集团微笑海岸旗下珠海东澳岛万豪度假酒店正式开业,为珠海增添一处集浪漫体验和自然美景于一体的海岛休闲度假旅游打卡胜地。 

2024年6月,珠海海岛用电全部实现“陆岛同价”,高质量打造“产业兴旺、生态宜居、岛风文明、生活富裕、治理现代”的现代化特色海岛镇,是打造“百千万工程”海岛新样板的一项重大举措。 

2024年7月,珠海市桂山岛风景区入选国家4A级景区。珠海万山区以桂山岛为示范样板,“广东省最美海岛”和“百千万工程”海岛新样板。 

2024年12月,珠海首台渔旅综合平台“海之舱”正式开工建造。该平台不仅具备先进的养殖功能,还拥有丰富的海上休闲旅游配套设施,将依托海洋牧场打造“海岛酒店度假+海上渔旅休闲”的全新文旅模式。 

03 

海运新航线 

珠海2024年海洋经济成果斐然。前三季度海洋生产总值同比增长6.8%,海洋经济延续稳中求进发展态势。其中,海洋交通运输能力不断提升,几条海洋经贸新航线和海岛旅游新航线的出现,为珠海海洋经济注入了“乘风破浪”的新活力。 

2024年10月,“珠海—海南—东南亚”内外贸同船+国际班轮航线开通。这条航线成为连接珠海、海南和东南亚的重要枢纽,有力推动区域内贸易和经济发展,对珠海深度融入全球物流供应链产业合作,加快推进开发开放建设具有重要意义。 

2024年12月,“珠海高栏港—巴西”航线成功首航,正式打通珠海与拉美地区经贸合作新通道、开啔远洋经贸新篇章,为珠海经济技术开发区深度融入全球产业链供应链注入了新的强劲动力。 

2024年8月,“东莞虎门港澳客运码头至珠海桂山岛、东澳岛”航线正式开通,该条航线是首条东莞市直航珠海海岛的水上高速客运航线,大幅缩短了两地的交通时间。 

国庆节前夕,在全省旅游发展大会上,一条涵盖汕头、汕尾、珠海、江门、阳江、茂名、湛江等地海岛的“海上风光跳岛游”线路亮相,为游客提供了更为丰富的旅游体验,也为珠海经济注入了新的活力。 

珠海三角岛 

04 

蓝色粮仓 

珠海水域滩涂资源丰富,渔业发展条件得天独厚,渔业是全市农业经济的“压舱石”。2024年以来,珠海不断建强耕海牧渔产业链,打造海洋渔业新模式。“格盛1号”、“伶仃牧场”、蓝色海发种业园等多个“蓝色粮仓”应运而生。

2024年6月,“格盛1号”正式交付投产,标志着该项目实现“开工即投产”的完美落地。8月,“格盛1号”首次投苗20余吨,此后“格盛1号”将新能源发电、环境检测和视频监控等现代化渔业生产系统应用到日常饲养中。 

2024年10月,珠海海发蓝色种业产业园(飞沙村基地)正式交付投产。这是华南地区最具规模的海水种业产业园之一,促进形成繁育养殖全产业链生态圈,全力实现“珠海苗”支撑“珠海渔”。 

05 

低空经济海洋新场景 

2024年,被称为“低空经济元年”,这一年,珠海加速布局低空经济,蓄势“起飞”。无人机航线首飞、半小时从海岛速达市区、海鲜从“浪尖”直飞“舌尖”……“低空经济+蓝色粮仓”等创新场景,为珠海海洋高质量发展注入新动力。 

2024年7月,全国首条海岛公共物流无人机配送航线在珠海万山区首飞成功,标志着全国首条海岛公共物流无人机配送航线首飞成功,也标志着粤港澳大湾区海岛低空公共物流航线首飞成功。 

2024年9月,桂山岛低空运营中心建成,为未来万山群岛的低空运营提供了重要支撑。10月底,“桂山岛—东澳岛”“桂山岛—外伶仃岛”两条低空物流航线开通,极大缩短了海岛与市区的物流距离。 

2024年10月,第三届珠海年鱼产业发展大会在桂山岛举行,一批白蕉海鲈由大型无人机搭载,从“格盛一号”深远海智能养殖平台顺利起飞,全程仅用10分钟便抵达会场,实现由海洋至陆地的配送,生动展现了“低空经济+蓝色粮仓”的全新应用场景。 

06 

蓝色金融 

海洋经济的高质量发展离不开金融“活水”的支撑。2024年以来,珠海持续发展金融支持海洋行稳致远。召开“蓝色金融”座谈会,实施“金融+海洋”工程,鼓励全市各金融机构面向涉海企业和项目创新提供优质海洋金融产品服务,全省首创海洋牧场装备登记备案,设立珠三角首家海洋支行,积极引导全市金融要素、资源加快导入海洋产业,提供全链条、全方位的金融服务,创造金融赋能“蓝色粮仓”的“珠海样板”,走出珠海蓝色金融的创新之路。 

7月18日下午,珠海市海洋发展局召开全市“蓝色金融”座谈会,邀请我市19家金融机构和12家现代渔业企业代表出席座谈会,共同探索蓝色金融的新模式和新路径,加快金融赋能海洋新质生产力,为我市“深蓝经济”注入“金融活水” 

2024年11月,“格盛1号”获得广东省首本现代化海洋牧场装备登记备案证,现代化海洋牧场设备有了自己的“身份证”。海洋牧场装备备案登记为企业注入强劲的金融动力,有效解决担保难、融资难的问题,推动海洋经济高质量发展。 

2024年12月,珠三角地区第一家海洋支行正式揭牌并发放首笔贷款。这是珠海以改革创新为抓手,推动海洋经济高质量发展的又一重大举措。 

07 

国家级沿海渔港经济区 

2024年以来,珠海加快渔业园区布局“升级”。印发实施《珠海渔港经济区建设规划(2021-2030年)》,投入超10亿元建成全国示范性现代化渔港——洪湾中心渔港,4月,洪湾中心渔港成功入选国家级沿海渔港经济区试点公示名单。 

未来,洪湾中心渔港将进一步拓展产业规模,增强渔港辐射功能和带动作用,促进水产品仓储加工、冷链物流、水产品贸易和休闲渔业等渔业经济综合发展,成为陆海岛统筹的发展典范和大湾区重要的水产品生产与集散中心,助力珠海打造集现代渔业生产、滨海旅游和海洋生物科技等为特色的现代渔港经济区。 

08 

海洋生态文明 

海洋生态文明建设是生态文明建设重要组成部分。党的十八大以来,在习近平生态文明思想的指引下,珠海全面推进海洋生态保护修复工作,加快建设安全、生态、健康、韧性、活力的“美丽海湾”“和美海岛”,休渔期间实行“最严格的伏季休渔制度”,举行增殖放流活动,有效提升海洋生态系统质量和稳定性,不断拓展优质亲海空间,促进人海和谐,对推进“海上新广东”和海洋强国建设发挥了重要作用。 

2024年6月,珠海市海洋发展局在洪湾中心渔港举行了2024年“全国放鱼日”增殖放流活动。此次共放流黑鲷鱼苗33.5万尾、黄鳍鲷鱼苗48万尾、斑节虾苗2280万尾。 

海洋伏季休渔制度是我国涉及人口最多、持续历史最长、影响力最大的渔业资源养护管理制度。为期三个半月的南海伏季休渔期间,珠海市应休渔船890艘,在珠海停泊休渔的港澳流动渔船166艘。2024年8月16日,珠海市洪湾中心渔港举行2024年南海伏季休渔开渔活动,渔船再次扬帆出海。 

2024年10月,“2025年广东珠海市海洋生态保护修复工程项目”成功入选全国海洋生态保护修复工程项目,获得中央财政资金3亿元。珠海万山海洋开发试验区持续致力于打造“生态和美、宜居宜游”的全国示范和美海岛,争做“广东最美海岛”。 

2024年12月,珠海万山海洋开发试验区以珠三角核心区第一名的成绩跻身“绿美广东竞芳华”全省十强。珠海万山海洋开发试验区持续深入推进绿美生态建设,以“一岛一特色、一宿一景观”发展理念,打造山海相连、蓝绿交织的生态景观,拓展亲山傍海、和谐共生的自然格局,建设通山达海、色彩多样的魅力绿美空间。 

09 

中华白海豚

中华白海豚是我国现存唯一的国家一级保护水生野生动物,也是唯一以“中华”命名的海洋豚类,被誉为“海上大熊猫”。目前珠江口海域生活着超过2000头中华白海豚。珠海海域是中华白海豚分布最密集的区域,是珠江口中华白海豚保护的“主场”。2024年,中华白海豚成为珠海城市吉祥物,双方开啔了更为紧密的“双向奔赴”、共生共荣之旅。 

2024年8月30日,珠海市十届人大常委会第二十八次会议经审议表决,通过了《珠海市人民代表大会常务委员会关于确定中华白海豚为城市吉祥物的决定》,确定以中华白海豚作为珠海的城市吉祥物。以立法形式确定城市吉祥物,在全国尚属首次。

《决定》明确,珠海市人民政府应当设计城市吉祥物的视觉化形象,并做好该形象相关知识产权保护工作。鼓励市民和企业事业单位及其他组织积极推广和免费规范使用珠海市人民政府设计的城市吉祥物视觉化形象,形成全民宣传效应,营造全城海洋氛围,打造政企合力宣传矩阵,共同宣传中华白海豚故事。 

2024年4月,广东、福建、广西、海南四省(区)检察机关齐聚广东珠海,召开中华白海豚保护工作研讨会,成立中华白海豚保护检察联盟。联盟检察机关共同发布《守护中华白海豚 维护生物多样性》检察倡议书,加强在公益诉讼线索移送、案件管辖、跨区调查、信息共享等多方面开展交流协作,共同维护区域海洋生态可持续发展。 

2024年12月,广东珠江口中华白海豚国家级自然保护区管理局与珠海市海洋发展局共建的首个“守护中华白海豚示范学校”正式交付使用。 

“守护中华白海豚示范学校”的建立,不仅是对中华白海豚保护的历史延续、执着坚守,也是提升珠海市民对城市文化符号的认同感和归属感、凝聚城市文化共识的有效方式,更是携手“海上国宝”打造美丽中国的城市典范、共建美好生态的全新承诺。 

10 

海洋管理机构改革 

2024年以来,珠海创新优化海洋管理体制机制,以新一轮机构改革为契机,组建珠海市海洋发展局,明确珠海市海洋综合执法支队管理职责,优化万山海洋开发试验区管理体制机制,再加上此前组建省内第一家地级市海洋集团,形成“三海联动”发力海洋发展格局,以科技创新引领现代海洋产业体系建设、加快培育发展海洋领域新质生产力,构建海洋治理新格局,以全新的势态跑出全新的海洋“加速度”。 

2024年4月,珠海市海洋发展局正式挂牌。挂牌成立以来,珠海市海洋发展局坚持以习近平新时代中国特色社会主义思想为指导,紧密围绕省委“1310”具体部署和市委“1313”思路举措,以珠海建设海洋区域中心城市为目标,坚定不移做好经略海洋这篇大文章。 

2024年,珠海迎来新一轮的区域改革,根据调整方案,万山海洋开发试验区回归鹤洲一体化运作前的管理方式。作为全国首个地方性海洋开发试验区,万山海洋开发试验区牢牢把握“百千万工程”和“海上新广东”建设机遇,坚持改革开放,聚焦重点发力,做好做足海的文章,打造“产业兴旺、生态宜居、岛风文明、生活富裕、治理现代”的现代化特色海岛镇。 

2024年7月,珠海市海洋综合执法支队正式挂牌。支队挂牌成立以来坚持践行“执法+服务”理念,在珠海打造“海上新特区”的新征程上贡献执法力量。

越南河内景点介绍_越南必去旅游景点有哪些

越南河内景燕窝专营店点介绍?

河内是越南的首都,虽然不是繁华大都市,但个人觉得能玩的景点还是挺多的。

1.巴亭广场。这相当于我们的天安门广场,当然面积没有天安门广场大,这里可以参观胡志明的陵墓、胡志明博物馆,周围则是办公的楼房,基本是法国风格的建筑,然后附近也有他们古代的皇城,这一带可以看的地方还挺多的。

2.三十六行街。在还剑湖边上,还剑湖也是河内的一个大景点,平时有很多当地人或者游客在湖边休闲散步的,然后三十六行街则是河内的老街,非常非常老的街,甚至可以说是破旧,但是这里卖东西的很多,吃的用的玩的,还有各港口燕窝店种小玩意手工艺品等应有尽有,很热闹而且非常有越南特色,边逛边找好吃的还挺惬意的。

3.河内啤酒一条街。其实也是在三十六行街里面,但是这里要特别说一下,啤酒街的特色是,每到晚上的时候,这里会非常热闹,街上有高端的酒吧,也有路边露天的酒吧,露天酒吧其实就是一些人在街边放一个啤酒桶,大概一块钱人民币左右可以买一杯啤酒喝,你是站着喝也好,走着喝也好,想要坐着喝也行,坐着的话,就是端一个小塑料凳在街边坐下,然后跟朋友变聊天边喝,有些人一杯啤酒可以喝一晚上,店家也不会赶你,大家都其乐融融的,氛围非常好,所以这里也有很多欧美的游客喜欢,晚上见到本地人以及欧美游客的数量几乎是一样多的。

4.河内大教堂。据说是仿巴黎圣母院兴建而,非常有中世纪古风,里里外外都非常庄严美丽,我第一次参观这种风格的大教堂,还挺吸引人的,主要是教堂附近也有很多咖啡店,逛累了可以在边上坐着喝咖啡吃点心,也挺不错的。

越南必去旅游景点有哪些?

越南,位于中南半岛东部,国土呈S形,南北共长两千公里,北与中国接壤,西与老挝、柬埔寨交界,东面和南面临南海,海岸线长3260多公里。

越南国内的旅游业已经相当成熟,每年数以百万计的来自全世界各个地方的游客,来到这个融合了东方神秘色彩和法国浪漫风情的国家。当地的名胜古迹不少,而且融合了东方的神秘色彩和法国的浪漫风情,加上农产品燕窝专营店丰富,各种水果非常便宜,物价指数也低,是非常值得一游的。

越南旅游好去处:

1、芽庄

芽庄是越南中部的一座海滨城市,是一个著名的海滨旅游胜地和港口城市。芽庄市是越南众多滨海城市当中一个较为僻静的海边小城市,与海上七大奇观的下龙湾相比较,芽庄的恬静内敛渐渐受到更多外国游客的关注。

芽庄位于河内到胡志明市的Open Tour线路上,游客来去非常方便。庄附近的海域是越南有名的渔场,这里每年可以出海捕捞的良好天气多达250天以上。附近福海地区为重要文化遗址。河口外有岛屿作屏障,空军基地,盛产鱼、虾和燕窝。

2、河内

河内是越南的首都,全国第二大城市。与充满法国情调的胡志明市相比,河内受到更多来自中国的影响,无论是古建筑上的汉字牌匾,还是无处不在的生活细节,都令中国游客感到几分亲切。河内地处亚热带,因临近海洋,四季如春,降雨丰富,花木繁茂,百花盛开,素有“百花春城”之称。

3、岘港

岘港是越南最重要的海港之一,曾经被法国人取名为“土伦”,在越南有着重要的军事战略地位。当年法国军队就是从这里入侵越南,1965年,美军也是由此登陆开始越战的。美国《国家地理》评岘港为人生必到的50个景点之一,认为它是现代文明与自然的完美结合。此外,岘港是世界上著名的六大海滩之一,在欧美人眼中,是难得的度假胜地。

海棠湾位于海南省三亚市东北部海滨,距三亚市区28公里,南面与亚龙湾国家旅游度假区毗邻。与亚龙湾、大东海湾、三亚湾、崖州湾并列三亚五大名湾。海棠湾风光旖旎,沉寂异常。与亚龙湾、大东海燕窝专营店相比,这里还没有染上城市的喧嚣与繁闹。

神州半岛海湾碧波映照,轮廓柔和宜人,沙白滩阔,加之奇岩怪石的点缀,景色格外迷人。目前岛上已发现泉眼八处。神州半岛最奇的还是千奇百怪的石头。离海岸几十米远的石岛,是大自然鬼斧神工的杰作。公鸡石、钓鱼石、观鱼石、乌龟石等等,随着潮涨潮落,变化万千。

神州半岛的背部为风平浪静的内海,其海岸线曲折,狭宽不一,沿岸植物郁郁葱葱,倒影水中,仿佛人间仙境。洁白松软的沙滩曲折悠长,伸向远方的港门港口;浅水岸边,海浪的泡沫冲洗着斑斓的贝壳;海湾深处的渔场,渔舟如梭,网群如云,在海滩上拉绳,收网时生蹦活跳的鱼虾在阳光下银光闪闪;深海处,一群群的渔船在追捕马鲛鱼、带鱼、西刀鱼、墨鱼等,观赏赏渔场繁忙的生产景象,令人别有乐趣。

海口旅游必去景点

海口风光秀丽,名胜古迹众多,旅游景点相对集中,海口必游景点有:骑楼老街、假日海滩、万绿园、雷琼海口火山群世界地质公园、海口钟楼等景点。

一、海口必游景点

海口必游景点——骑楼老街

海口骑楼老街,是海口市一处最具特色的街道景观。其中最古老的建筑四牌楼建于南宋,至今有600多年历史。海口老街主要分布在市区得胜沙路、新华南路、中山路、博爱路及解放路一带,街道两旁是近百年历史的充满南洋建筑风情的骑楼港口燕窝店。尽管已过去了百余年,走在海口的骑楼老街上,仍能欣赏到街边建筑的美。

海口必游景点——假日海滩

假日海滩位于海口市滨海大道西延线庆龄大道北部,东起西秀海滩,西至五源河口,长约6.2公里,距海口市中心区11公里,绵长的海滩边是一望无际的琼州海峡,常有帆船出没,规模庞大的露天海鲜烧烤广场晚间热闹非凡,附近有贵族游艇会及多家豪华酒店,沿着假日海滩,有宽广干净的滨海大道,在这里散步吹风很是舒服,此外,晚上可以在海滩东边的剧场看《印象·海南岛》的实景演出。

海口必游港口燕窝店景点——万绿园

万绿园位于海口市滨海大道北侧的填海区,东连海口市体育馆,南隔滨海大道,北对琼州海峡,是海口市最大的、开放性的、热带海滨生态林式风景园。万绿园是海口最大的集植物旅游观光、晨练运动于一体的大型公园,每天万绿园里人来人往。在万绿园里早晨练、晚散步,是许多海口市民必不可少的功课。你也可在早晚来万绿园转上一转,当一次海口土著。

海口必游景点——雷琼海口火山群世燕窝专营店界地质公园

雷琼海口火山群世界地港口燕窝店质公园是我国唯一的城市滨海热带火山,也是海南省第一家世界级旅游景区。公园地质遗迹主体为40座火山构成的第四纪火山群——远古的琼北火山爆发,在这里遗留下世界上保存最完整的火山群,似一座座天然的火山地质博物馆。

海口必港口燕窝店游景点——海口钟楼

海口钟楼位于海口市区北部长堤路海口儿童公园内,面向入海口,南面中山横街与中山路相对,北临海甸溪,依街傍水,姿影绰约,景致十分幽雅。海口钟楼被列为海口八景之一,它历史悠久,最早是为适应对外通商而建立,已成为海口的最重要的标志性与象征性建筑物之一。钟楼对面那些古旧斑驳的建筑群就是著名的骑楼老街。

1、到骑楼小吃街享文化美食两大盛宴

小吃街将海南骑楼、琼剧、黎锦、黄花梨、海捞瓷、小吃等六大海南文化元素融为一体,以其独特的装饰风格和传统的风味小吃得到了游客赞赏,闻名海内外的文昌鸡、嘉积鸭、东山羊、和乐蟹,还有海南粉、抱罗粉,陵水酸粉、琼海粗粮、定安肉粽灯光小吃都能在这里吃到。

燕窝专营店2、文明中路品地道海南鱼煲

海口人一直是喜欢吃鱼的,但喜欢把鱼炖了汤或者和别的菜混合一起来煲。所以海口老底子最有名的菜,就是鱼煲,哪一条老街巷弄都有,做法主要有琼式鱼煲和川味鱼煲两种。推荐位于文明中路的第十三小学门口旁的鱼煲店,这家是加有酸芋梗和酸笋的传统海南做法,原汁原味,尤其酱料是一绝,再配上香喷喷的鸡饭更是让人吃了也不腻。

3、到“印象海南岛”燕窝专营店聆听大海之歌

到海口,一定要抽时间去欣赏《印象海南岛》,这是一场以大海为实景、以大海为主题的实景演出。演出以音乐剧为主,通过独特的魔幻水升降舞台将海南岛独有的热带休闲文化表现地淋漓尽致。梦幻舞台、浪漫夜空映照下,剧场每晚还燃放激情焰火,水上烟花与外沙滩烟花齐放,与《印象海南岛》的演出一同照亮黄金西海岸灿烂的夜空。

4、漫步万绿园,燕窝专营店悠享慢生活

万绿园拥有草原般的广阔,上千亩的草坪一望无际。沿海的马路边,更是有近万棵以椰子树为主的热带树木,婆娑起舞,马路长达数公里,适合跑步、练拳、舞剑、放风筝、港口燕窝店骑双人自行车等等,每天清晨和傍晚,随处可见锻炼的身影。

5、到板桥路燕窝专营店吃海鲜大排档

海口市有两个板桥路,靠近青年路的板桥路是值得一去的'好地方。那里有很大的海鲜市场,你可以挑选感兴趣的海产港口燕窝店品,旁边有很多大排档老板,可以帮忙加工,他们会很热心地陪你去挑选海鲜,帮你讲价。加工费的价格是由你食材的烹调方法单价与重量相乘来计算的。

四、海口八景

海口八景到底哪里来的呢?根据海口市志记载,1990年海口评出的海口八景。尽管现在看来,有些确实非常平凡,然而,它却曾经象征了海口的一个形象,还是值得怀念的。

海口八景:古钟新声(钟楼)、平湖秋月(东湖、西湖)、椰风海韵(西海岸带状公园)、圣祠叠彩(五公祠)、琼崖丰碑(中共琼崖一大旧址)、金厦银灯(海口国际金融大厦)、粤东正气(海瑞墓)、勋亭流芳(李硕勋烈士纪念亭)。

四、海口旅游小提示

1、海口旅游消费指数

比起北京、上海、深圳等地,海口的消费水平偏低,餐馆消费从人均30到150不等,也有人均不到15的路边摊,各种档次的美食,任你选择。住宿方面,高档酒店价格区间为498到1400不等,中档酒店价格区间为258到398不等,经济型酒店价格区间为105到198,青年旅社价格区间为78到100不等燕窝专营店,适合各类需求的消费者。

2、海口旅游交通

去往海口有多种交通方式可以选择,飞机、火车、长途大巴以及轮船。

相比较而言,远程旅客选择飞机前往是最为快捷的方式。

周边的游客就多多选择火车和大巴吧~

当然,广州和湛江的旅客首选轮船绝对没错哦!

要想了解一个城市,一定要坐公交试一试哦。海口市内交通发展比较完善,公交每站间距都比较短,各个公交线路基本可以到达各个景点。

3、海口旅游住宿

海口的酒店、旅馆数量众多、档次齐全。即使在旅游旺季,当三亚等地的宾馆不堪重负的时候,在海口仍可以找到满意的住处。 商务人士可以考虑海口的各大星级酒店。如果是学生族去旅行的话,住在青年旅舍无疑是上佳的选择,不仅安全卫生而且便宜。海口的住宿地点可以分为市中心、西海岸和海甸岛三大区域,都各有特色。Tips: 价格方面,除节假日外,平时都可以打折,甚至可以低到对折。但酒店门市价的季节性变化非常大,如果想省钱,就避开旅游旺季。

4、海口旅游美食

海口菜的技法与粤菜差不多,但却以椰味见长,比如椰奶鸡、椰奶燕窝盅等。海南四大名菜(文昌鸡、加积鸭、东山羊、和乐蟹),以及石山扣羊肉、曲口海鲜、四宝琼山豆腐、斋菜煲这些在海口都能吃到。还有各种虾、贝、鱼,应有尽有,让人大饱口福。

早茶——大清早,海口市内各大小茶楼、餐厅、酒店就已座无虚席,热闹的早茶景象会直到10点多。特别推荐骑楼老街——博爱路的水巷口街,是海口地方饮食店聚焦处,也是吃早点的理想场所,海南腌粉、牛腩饭、猪脚饭、鸡饭和鸭饭、煎粽、煎饼、安定粉汤很受欢迎,海南人还会在鸭饭上面浇一种汁,使得鸭皮在脆的同时鸭肉又很滑口。

夜市——海口是一座不折不扣的不夜城,而宵夜则成为不夜海口中最为重要的部分:啤酒、小菜、海南粉、各类烧烤、特色米线、火锅、清补凉、海鲜、小黄牛、乳羊等等海口宵夜食品的品种之全,真应了一句话:“只有想不到的,没有吃不到的。

5、海口旅游购物

在海口不妨逛逛当地的土特产商店,在那儿买些兴隆咖啡、胡椒粉、椰糖之类的东西带回去,无论是自己吃还是送人都是很不错的,尤其是海南的胡椒粉,非常香,很值得买。这些土特产一般都由厂家直销,因此价格不会很贵。另外,海鲜干货也应该买一点,大东门市场是海鲜干货的集散地,可以在这里买一些美味的红鱼或鱿鱼干。

6港口燕窝店、海口最佳旅游时间

10月到翌年5月是海口旅游的最佳时节。这期间海口风平浪静,气候怡人。北方酷寒来袭时,这里却暖风和煦。此时,海南还有农历“三月三”期间的“椰子节”和农历二月初九至十九的“军坡节”。一个是融合以海南椰文化和黎苗“三月三”民俗为特色的国际盛会,一个是在海南相传1300多年的乡情民俗。

7、海口旅游穿衣指南

海口全年的平均温度都在20℃上下,即使在冬季,相对北方的天气,也是温暖舒港口燕窝店适的,只需准备简单的外套和T恤即可。夏季多雨,请备好雨具。

尽管海口全年的平均温度都在20℃上下,但每年5至10月是海口的雨季,尤其9月,是降雨高峰期,游客出行会有不便。此外,海南岛的东部被称为“台风走廊”,因此夏秋旅游别忘记收听气象部门发布的台风警报。遇上强台风侵袭,海口交通会受影响,可能航班会取消,水路会停运。

海南旅游攻略必去景点有哪些?

1、玉蟾阁

文笔峰位于海南省定安县的中部,它平地拔起,山色秀美,为海南省著名景点之一。文笔峰从山腰至山顶主要由玄武岩和片理岩组成,山上植被茂盛,山顶常有云雾缭绕。

文笔峰,又名文豪岭。明朝至今,龙梅村还有一座为纪念王弘诲所建的富有海南建筑特色的明代牌坊,建筑设计造型古朴雄厚。

2、箭毒木

海南热带植物园包括海南兴隆热带植物园、海南儋州植物园两个园区。海南热带植物园是我国热带地区十分重要的热带作物科研教育基地、试 验、示范基地和物种基因库,它填补了海南观光农业及生态旅游的空白,成为我国热带农业考察重点单位。

3、金沙滩

玉带滩是一条自燕窝专营店然形成的地形狭长的沙滩半岛,位于海南博鳌,外侧南海烟波浩淼一望无际,内侧万泉河、沙美内海湖光山色,内外相映,构成了一幅奇异的景观,滩上更生活着国家二级保护动物玳瑁。可惜后来由于台风,沙滩被拦腰折断成两截。

3、海洋文港口燕窝店化馆

分界洲岛位于海南岛的东南海面,距海南岛最近海岸约1.2海里,乘船过渡单程约15分钟。小岛以东北向西南长条状横卧在蓝色大海中,远看呈马鞍形,面积约0.41平方公里,海拔最高为99米。

海南分界洲岛旅游区自2004年5月1日开业,2013年1月被国家旅游局评定为港口燕窝店国家5A级旅游景区,是中国首家海岛型国家5A级景区。

3、热带天堂森林公园

亚龙湾是中华人民共和国海南省三亚市东郊的一处优质热带海滨风景区,距离市中心区约10公里。海湾以中心有野猪岛为中心,南有东洲岛、西洲岛,西面有东排、西排,可开展多种水上运动。岸上林木郁郁葱葱。

冬季这里的气温27摄氏度,水温20摄氏度,是一处理想的冬季避寒和休闲度假胜地,号称“东方夏威夷”。

4、大港口燕窝店东海

大东海位于三亚市的榆林港和鹿回燕窝专营店头之间,是三亚的一处标志,被国家旅游局评为“中国四十佳景”之一。

大东海三面环山,海岸线长2.9公里,一面大海,一排排翠绿椰林环抱沙滩,蓝天、碧海、青山、绿椰、白沙滩独特之美博得海内外游客的赞叹。大东海沙滩平坦细软,缓缓延伸,长达千米。“水暖沙白滩平”早已使大东海斐声海内外。

5、天涯海角

天涯海角游览区,位于三亚市天涯区,距主城区西南约23公里处,背对马岭山,面向茫茫大海,是海南建省20年第一旅游名胜,新中国成立60周年海南第一旅游品牌,国家AAAA级旅游景区。景区海湾沙滩上大小百块石耸立,“日月石”和“南天一柱”突兀其间。

1,蜈支洲岛

蜈支洲岛坐落在三亚市北部的海棠湾内,北面与南湾猴岛遥遥相对,南邻美誉天下第一湾的亚龙湾。蜈支洲岛距海岸线2.7公里,方圆1.48平方公里,呈不规则的蝴蝶状,东西长1400米,南北宽1100米。

港澳台买什么特产最好

港澳台值得就是我国的香港、澳门和台湾,这些地区由于近代特殊的历史愿意,所以显得格外特殊,这三个地方也是著名的旅游胜地,下面给大家分享详细的攻略。

澳门大三巴前的手信街

我信手捎来,你喜出望外。“手信”二字,正是港澳台地区对伴手礼的叫法。

来北京,买份糕回家才地道。那去港澳台又可以买些什么?

港澳台,手信种类多,服务态度好,不愁买不回得体的礼物。这份手信业的繁荣,与港澳台被殖民的历史有关。彼时时局动荡,亲眷别离,隔海相望,唯有家中的特产,能一解远洋的思乡之情。

从太平山顶俯瞰香港。

香港与澳门的文化多源自于广东省,而台湾则源自于福建省,所以手信的种类与口味也各有千秋。随着国庆假期的结束,上班日子的临近(别再说了!),风物菌给你好好安利一下这三地值得买的手信。

澳门丨免费试吃吃到饱

澳门手信有近百年历史。1999年澳门回归,博彩业兴盛,让手信这个夕阳产业咸鱼翻身。一件件手信从最初的纯手工制作,到如今的机械化量产,满足了一批又一批的游客。

杏仁饼里有杏仁

杏仁饼,真香真酥脆。

早期,澳门以制作饼食的店家居多,杏仁饼作为闲时的配茶小点,最为常见。经过炭火的烤制,杏仁饼口感酥香,还带有一丝炭火气味。每咬一口,都能够看到大颗的杏仁粒,十分良心。早期兴盛的客运码头,迎送着商旅和移居海外的澳门人。饼店便在港口聚集,为他们远洋前携上一份家乡味道提供方便。

还属手工制作的杏仁饼好吃。

现在,澳门的饼店大多保留着制作炭火炉烤的传统手艺。手工杏仁饼推荐去澳门新马路的“最香饼家”购买,每天大概只卖1000个左右,又以酥香不粘牙的蛋黄杏仁饼为佳品。而钜记手信的杏仁饼种类繁多,像原颗杏仁、咸蛋黄味、海苔味等多种味道,送人显得大气。

被列入必买单中的“最香”杏仁饼、凤凰卷。

老婆饼里没老婆

现烤老婆饼,撩人又温存。

就像女子的内心,真正的老婆饼是温柔甜美的。老婆饼又叫冬蓉酥,由冬瓜作为馅料,味道微甜不腻,饼皮酥软,细细咀嚼能有软滑顺口的体验。相传是广州莲香楼的潮州点心师傅改良了自己老婆制作的冬瓜角,变成了冬蓉酥,因此起名为老婆饼。

晃记饼家的菜单,都想点

澳门晃记饼家的招牌就是老婆饼。这家店有着上百年的历史,现在坚持每天手工现做。刚出炉的老婆饼外皮更酥,味道也不像流水线生产的老婆饼腻味,甜度刚好。即使冒着烫嘴的风险,也忍不住咬上一口。除了响当当的老婆饼,广东名点鲍鱼酥、核桃酥、肉切酥和鸡仔饼等,都能在晃记买到,一站搞定。

可硬可软花生糖

花生糖和姜糖,在百年之前就被钜记手信的伙计放在小推车上沿街售卖。澳门花生糖的优秀之处在于它柔软不粘牙。最早黑芝麻口味的花生糖里,烘焙过的芝麻香香脆脆,花生颗粒饱满。发展到了现在,花生硬糖还分支出了软糖,夹着椰丝的、海盐的和白芝麻的,琳琅满目。

椰丝花生糖和黑芝麻花生糖,现做现卖,保质期短。

钜记的每一个门店都有现做的试吃,服务员的态度也非常诚恳,就算不买也能大饱口福。

猪肉脯,原切的更好

澳门的猪肉脯讲究口味多样,以现吃为主。

猪肉脯其实是新加坡的特产,在1928年传入广东,并由澳门手信业发扬光大。澳门每家手信店除了点心糖果,猪肉脯占去半壁江山。猪肉脯的摊位,光靠着甘香就足以常年聚拢大批客人。一眼望去,成堆成垛的猪肉脯整齐码放,有原味、辣味、薄切和后切,还有牛肉脯可以选择。店员会为你用剪子剪下不同口味的肉脯免费试吃,机会可得好好把握。

现剪现尝猪肉脯。澳门服务员的态度很热情,请把拘谨抛开,多试不同口味。

作为手信,原切的猪肉脯是首选。不同于猪肉搅碎后拼成的猪肉脯,原切猪肉脯是用一整块肉腌制风干制作,未被破坏的肉纤维为口感做背书,原汁原味,独具匠心。当然,价格也会稍贵一些了。

别有韵味的香港铜锣湾登隆街。

买鞋买包买奶粉,是大多数人去香港的任务。香港美食能开出怎样的花样?详见《带着香港胃,扫街香港夜》。同样是深受广东文化的影响,香港的手信并不像澳门那么多,蛋卷、杏仁饼和鸡仔饼,有名的几样也与澳门稍有重合。但只要细究,就能一窥香港人别样的小心思。

为孩子发明的燕窝糕

清淡饮食,靓汤粥水,都是香港人的养生秘籍。燕窝羹常见,但燕窝做的糕点你见过吗?

陈意斋的燕窝糕,古朴的包装值得玩味。

1927年,主打健康零食的陈意斋在佛山诞生。当时,陈老板看身边的小孩吃的零食都不健康,所以就发明了燕窝糕。这一道糕点采用真正的燕窝制作,软糯中带着点韧劲儿,甜味清雅。

选料天然,保质期只有7天而已。如今,香港人一提起燕窝糕,就会想到陈意斋。你若用它做为手信,尤其是用来孝敬长辈,会显得尤为高级。

陈意斋里其他的养生零食。

陈意斋的小吃不只燕窝糕一种。川贝杏仁露、川贝旧陈皮、甘草榄和柠汁冬姜等零嘴,都把养生健康的理念一一落实。

把凤梨酥做成曲奇

香港人热爱饮茶食点心,很多人来到香港都会带一些诸如蛋卷的酥点回家。但是自从Jenny

bakery(珍妮曲奇)进驻,香港传统的酥点经历了不小的挑战。你也许对珍妮曲奇感到陌生,但港剧中常出现的熊仔曲奇饼干,正是他家的招牌。

珍妮曲奇熊仔饼的铁盒包装给人感觉很温暖。

原味牛油花、咖啡牛油花、脆牛油还有提子燕麦,是珍妮曲奇最经典的4mix。

这曲奇可不是传统味道,而是现代新式口味。老板Jenny是一位来自马来西亚的华侨,早在1989年便随丈夫移居香港。Jenny制作曲奇的好手艺,传颂在香港南部赤柱妇女界,街坊邻里都特别喜欢Jenny亲手制作的黄油曲奇饼干。

原珍妮曲奇的价目表,你想吃哪个?

珍妮曲奇,有着特别浓厚的黄油香气,入口松软,用口腔的温度就能融化。最特殊当属对凤梨酥进行改良,将传统凤梨酥的酥皮换为曲奇,包裹着真材实料的凤梨果蓉,酸甜酥松。因为是相当热门的品牌,所以山寨它的人很多。你寻到尖沙咀和上环去,那里才有珍妮曲奇正版店铺。

灯火通明的台北101大楼。

在台湾,手信也被叫做“丹禄”。相传,台湾早期人民生活穷苦,逢年过节给亲朋送的伴手礼都是平日辛苦攒下的积蓄,就像炼丹一样的艰辛。虽说现在在台湾准备手信已不用呕心沥血,但当地好味的特产依然包含着丝丝人情味。

凤梨酥很好吃,但我选芋头酥

加入麻薯馅的趸泰牌的芋头紫晶酥。

芋头酥,可不比热门的凤梨酥逊色。亚热带季风气候和热带季风气候并存的的台湾,适合各种热带水果及蔬菜的种植。从台湾的平地到海拔1000米的山区,都能找到芋头的身影,而且台湾芋头品种丰富,风味还各有不同。

芋头酥,台湾手信的“优等生”。

芋头卷、芋圆、芋头奶茶和芋头酥,台湾人吃起芋头来花样多多。其中芋头酥便于携带,所以在手信C位出道。和凤梨酥的软酥皮不同,芋头酥可是正儿八经的层层叠叠。

玫瑰花的紫色酥皮造型、台湾独特芋头品种制作的芋泥,再加上弹性十足的麻薯,口感的层次很丰富。在此,推荐你买趸泰牌的芋头紫晶酥,它家的芋泥香味突出,甜度适中,一家老小都可以吃。

猪肉纸,0.01版的猪肉脯

前有台湾媒体发文《台湾手信猪肉纸,连吃10天体重爆增1公斤》,后有小s在《康熙来了》中推荐,猪肉纸真有那么传奇?顾名思义,猪肉纸就是薄如纸片的猪肉脯。之所以这么薄,多亏了台湾独特的热风烘干技术。它在保留了猪肉脯甘香的同时,又最大程度地去掉其中的水份。

薄脆酥香,台湾猪肉纸。

在台湾迪化街的江记华隆商行,招牌香脆杏仁纸值得带回家——在猪肉纸中加入了美国大杏仁,坚果的香气能缓解猪肉的油腻感,口感还能变得像饼干一样脆,不能再满足了!

江记华隆的门面。夜深了也有人争相购买。

台湾的牛轧糖不粘牙

手信除了点心和肉脯外,糖果自然少不了。在11世纪的十字军东征时期,法国人将牛轧糖带回西方,并加入了坚果和梅干等进行改良,结果风靡上流皇室。在1956年时,台湾开始兴起牛轧糖,至今60余年这股热潮依在。在新北土城区,还建有牛轧糖博物馆。

台湾牛轧糖,许多普通的吃食,经湾湾的巧手后会变得很特别。

台湾牛轧糖的原料以甜度较低的海藻糖为主,更能够突出牛轧糖的奶香。因为用料的讲究,台湾牛轧糖吃起来不粘牙,香味真实。就算是不爱吃甜食的人也能吃得停不下来。想买牛轧糖手信,可以选择“糖村牌”。它的口味很多元,抹茶、咖啡和草莓等等,再特殊的少女心也能被满足,所以被人们称作“梦幻牛轧糖”。

景燕窝专营店点介绍

伶仃峰

外伶仃岛地势东西高,北和中间低,东部沿岸较陡,岛中央主峰伶仃峰高311.8米,爬上伶仃岛的主峰可以俯瞰全岛,山顶嶙峋怪石和飘渺云雾组成的美景引人入胜,天晴时可看到香港、桂山等港口燕窝店。

在城市呆久了,不妨来攀一下伶仃峰,感受环山慢跑或拾级而上的乐趣,俯瞰全岛的美丽风光,起个大早,登顶观日,登高燕窝专营店望远,静静地欣赏山水一色的极致胜景。

玉带环腰

玉带环腰位于外伶仃岛正面的两边,在山腰和礁石丛中,有一条精雕细凿的石阶小径,人称“情侣路”,在绿林和碧海之间蜿蜓,远远望去,如美人腰间玉带,故名“玉带环腰”。?

石阶小径沿着海岸蜿蜒,一边是俊秀的山峰,一边是清澈的海水。岸边大块大块的礁石,天气晴好,还可以站在礁石上近距离接触大海。看着远处的海浪涛涛、渔船往来,听着鸟叫虫鸣,适合放空思绪,缓步慢行,拾阶而上,见路即行,不经意间就能在静谧而喧哗间踏遍小岛风景。

港口燕窝店石景公园

石景公园依山傍水面海,除了怪石嶙峋和壮美风光,还可以朝拜“北帝晨钟”。在传说中“北帝”是统领所有水族的神仙,生长在海边的渔民普遍崇信北帝能为他们消灾解困,确保出海的一帆风顺,为此颇受拥戴。不少来自万山其它岛屿、珠三角各地的渔民、信众、游客纷纷上香祭拜,祈求平安。

伶仃沙滩

来伶仃沙滩戏水吧,港口燕窝店趁阳光正好,带上伴侣友人还有相机,记录美好的瞬间。

也可以选择适合自己的海上运动休闲一下,亦可体验从未有过的挑战。玩玩沙滩排球、海上摩托艇、冲浪香蕉船等,还可以体验海钓、潜水等娱乐项目,或者租船到临近小岛上游玩。

PS:沙滩边还有乘坐快艇环岛观光,从另一种角度观赏全岛,收费标准约150元/人,每年的休渔期禁止渔船出海期间除外。

伶仃湾

外伶仃岛上的优质沙滩,海水清澈见底、沙质细腻,保持着最原始的自然风貌。在伶仃湾,你可以游泳、徒步,在细腻的沙滩上发呆、吹海风、看大海,享受悠港口燕窝店闲惬意的假期。

拱桥渡水

外伶仃岛之行有什么好的攻略安排?

外伶仃岛旅行的攻略重点有:挑选旅游淡季前往,准备好防水的防晒霜以及提前联系预约当地渔船。在外伶仃岛吃海鲜、攀爬伶仃峰、在峰顶看日出,坐船出海等都是极具当地特色的旅行体验。

由于都是需要长时间驻留的室外景点偏多,去往外伶仃岛旅行要注意做好防晒措施。在外伶仃岛无论是赶海摸海鲜还是在海边看夕阳,都需要长时间停留在户外,因此防晒准备很重要。可以准备一些宽檐帽、披肩等硬防晒,防晒效果会更好。但如果天气过于炎热,就需要准备防晒霜了。防晒霜推荐防护能力比较强、倍数比较高的,不需要频繁补涂,不会影响游玩效率。

伶仃峰的日出壮美灿烂,是旅途中一定要体验的一部分。一般要提前观察好未来几天的天气,挑选一个没有阴雨的晴天,在日出的时间前留出一段爬山的时间,这样等到爬到山顶,就恰好可以看到伶仃峰日出了。紫金色的朝霞围绕着炽烈的太阳,景色壮观令人心旷神怡。在山顶环绕四周还可以俯瞰整座伶仃岛,放眼望去四周大海无边无际,几艘小燕窝专营店船点缀在平静无波的海面上,让人顿生渺小之感。如果想爬到山顶看日出要注意携带几个手电筒,否则山路崎岖比较难走。

在伶仃岛旅行,可以吃到很多肥美又平价的海鲜。如果在旅游淡季前往,海鲜的价格还会更低,并且都是从海中新鲜捕捞的鲜活海产。买来后可以找当地大排档或者民宿老板代为加工,加工费也很便宜。如果想亲身体验出海打鱼的乐趣,还可以提前预约好对外出租的渔船,自己驾驶着渔船出海,在海面上撒网捕鱼,又是另一种身临其境的乐趣。

此生一定要去一次的美丽小岛—珠海外伶仃岛,全攻略来了

外伶仃岛石奇水美,四季如春,优雅恬静闻名遐迩。是广东唯一能看到香港的海岛。民族英雄文天祥《过伶仃洋》的名句“人生自古谁无死,留取丹心照汗青”就诞生于此。

越南河内景点介绍_越南必去旅游景点有哪些

越南河内景点介绍?

河内是越南的首都,虽然不是繁华大都市,但个人觉得能玩的景点还是挺多的。

1.巴亭广场。这相当于我们的天安门广场,当然面积没有天安门广场大,这里可以参观胡志明的陵墓、胡志明博物馆,周围则是办公的楼房,基本是法国风格的建筑,然后附近也有他们古代的皇城,这一带可以看的地方还挺多的。

2.三十六行街。在还剑湖边上,还剑湖也是河内的一个大景点,平时有很多当地人或者游客在湖边休闲散步的,然后三十六行街则是河内的老街,非常非常老的街,甚至可以说是破旧,但是这里卖东西的很多,吃的用的玩的,还有各种小玩意手工艺品等应有尽有,很热闹而且非常有越南特色,边逛边找好吃的还挺惬意的。

3.河内啤酒一条街。其实也是在三十六行街里面,但是这里要特别说一下,啤酒街的特色是,每到晚上的时候,这里会非常热闹,街上有高端的酒吧,也有路边露天的酒吧,露天酒吧其实就是一些人在街边放一个啤酒桶,大概一块钱人民币左右可以买一杯啤酒喝,你是站着喝也好,走着喝也好,想要坐着喝也行,坐着的话,就是端一个小塑料凳在街边坐下,然后跟朋友变聊天边喝,有些人一杯啤酒可以喝一晚上,店家也不会赶你,大家都其乐融融的,氛围非常好,所以这里也有很多欧美的游客喜欢,晚上见到本地人以及欧美游客的数量几乎是一样多的。

4.河内大教堂。据说是仿巴黎圣母院兴建而,非常有中世纪古风,里里外外都非常庄严美丽,我第一次参观这种风格的大教堂,还挺吸引人的,主要是教堂附近也有很多咖啡店,逛累了可以在边上坐着喝咖啡吃点心,也挺不错的。

越南必去旅游景点有哪些?

越南港口燕窝店,位于中南半岛东部,国土呈S形,南北共长两千公里,北与中国接壤,西与老挝、柬埔寨交界,东面和南面临南海,海岸线长3260多公里。

越南国内的旅游业已经相当成熟,每年数以百万计的来自全世界各个地方的游客,来到这个融合了东方神秘色彩和法国浪漫风情的国家。当地的名胜古迹不少,而且融合了东方的神秘色彩和法国的浪漫风情,加上农产品丰富,各种水果非常便宜,物价指数也低,是非常值得一游的。

越南旅游好去处:

1、芽庄

芽庄是越南中部的一座海滨城市,是一个著名的海滨旅游胜地和港口城市。芽庄市是越南众多滨海城市当中一个较为僻静的海边小城市,与海上七大奇观的下龙湾相比较,芽庄的恬静内敛渐渐受到更多外国游客的关注。

芽庄位于河内到胡志明市的Open Tour线路上,游客来去非常方便。庄附近的海域是越南有名的渔场,这里每年可以出海捕捞的良好天气多达250天以上。附近福海地区为重要文化遗址。河口外有岛屿作屏障,空军基地,盛产鱼、虾和燕窝。

2、河内

河内是越南的首都,全国第二大城市。与充满法国情调的胡志明市相比,河内受到更多来自中国的影响,无论是古建筑上的汉字牌匾,还是无处不燕窝专营店在的生活细节,都令中国游客感到几分亲切。河内地处亚热带,因临近海洋,四季如春,降雨丰富,花木繁茂,百花盛开,素有“百花春城”之称。

3、岘港

岘港是越南最重要的海港之一,曾经被法国人取名为“土伦”,在越南有着重要的军事战略地位。当年法国军队就是从这里入侵越南,1965年,美军也是由此登陆开始越战的。美国《国家地理》评岘港为人生必到的50个景点之一,认为它是现代文明与自然的完美结合。此外,岘港是世界上著名的六大海滩之一,在欧美人眼中,是难得的度假胜地。

钦州周围有什么好玩的地方?

钦州市冬暖夏凉,气候宜人,依山临海。钦州风光明媚,古迹众多,文化灿烂,自古以来就成为桂南著名旅游胜地。她以其独特的优势和丰富的旅游资源,吸引了不少旅客。宋代文学家、诗人苏东坡曾"遨游钦灵";1891年,俄太子尼古拉仰慕冯子材抗法战功特来钦州访问;1909年,著名画家齐白石游钦州时,绘了荔枝图,并赋诗:此生无计作重游,五月垂舟胜鹤头,为口不辞劳跋涉,愿风吹我到钦州;1962年,戏剧家、诗人田汉来钦州游览时留下不少美好的诗句。  钦州旅游景点有30多处,主要有:三娘湾旅游区 、八寨沟旅游区 、冯子材故居、 刘永福故居、 大芦村民族风情、 六峰山 、五皇山、王岗山 、椎林叠翠 、麻蓝岛......等等。近年来,全市建造了一批星级宾馆和涉外酒店,成立了一批旅行社,制作了许多旅游工艺品,初步形成了食、住、娱、购配套体系,吸引了大批旅客前来观光。其中,钦州市内主要宾馆有:金湾大酒店、钦州宾馆、交通宾馆,银湾大酒店,白海豚国际酒楼等;主要的购物场所有:钦州市百货大楼、商业大厦、金湾商场、南珠商场,协胜,华润等。   钦州在明朝和清朝曾两次评过钦州八景,第二次评定至今已有300多年。现经钦州八景评审委员会审定,选出了钦州新八景。分别是:王岗春色、六峰缀秀、龙泾环珠、刘冯宝第、灵东浴日、麻蓝仙岛、越州天湖、椎林叠翠.
