珠港澳大桥

是世界建筑史上历程最长、投资最多、施工难度最大

也是最长的跨海大桥

堪称交通界的“珠穆朗玛峰”

被英国卫报评为“新的世界七大奇迹”之一

而这个“世纪工程”的背后

有着数量庞大的建设者

为之付出了辛勤的汗水

而能够参与港珠澳大桥的建设

也是建设者们的骄傲

不仅有位的工程师

是梅州大客家人蔡应广

还有一位重量级的人物

港珠澳大桥建造总工程师

卢永昌

是梅州梅县区的客家人

卢永昌，广东省梅州市梅县区桃尧镇人，中交第四航务工程勘察设计院有限公司总工程师、教授级高工。

1981年松口中学高中毕业，1985年，他从天津大学港口及航道工程专业毕业，进入中交四航院工作。

2006年，在职取得河海大学工程硕士学位。1999起，开始担任四航院总工程师，同年获得国务院特殊津贴。

2012年，荣获广东省劳动模范。担任总工以来，先后参与国内的广州港南沙港区系列工程、湛江港系列工程、汕头港系列工程、华南地区液化天然气接收站系列港口工程等，以及国外“一带一路”重要节点巴基斯坦瓜达尔港工程、斯里兰卡汉班托塔港工程、斯里兰卡科伦坡港口项目、喀麦隆克里比港口项目等。

“年少时的艰苦生活，磨炼了我的意志”

卢永昌出生于梅州市梅县区桃尧镇显朝村，家中还有一个姐姐、哥哥和妹妹。在村里念到初二后，便到桃尧中学念初三，由于不符合住校条件，每天他都要走1个多小时的山路才能达到学校。“6点左右就得起床出发了。”卢永昌说，学习之余，还要帮着家里干些农活，生活充满了艰辛。

“印象很深的是，有时候还要早上2、3点起床干农活。”回忆起往事，卢永昌笑着说道，也正是年少时的艰苦生活，磨炼了他的意志，教会了他吃苦耐劳，也在后来的学习、工作中不断激励他前行。

1978年，他升入松口中学，3年后，他以全校最高分425分考入天津大学，并选择了港口及航道工程专业，水运大师之路也就此开啔。“我是幸运的，虽然在小山村，但客家地区向来崇文重教，我的父母都也很支持我读书。”卢永昌说。

如今，卢永昌每年都会回乡看望父母，谈及家乡的变化，卢永昌表示，随着新农村建设的深入，村子的卫生环境也不断改善。

作为金柚种植大镇的家乡，他希望当地政府能进一步引导柚农加强金柚种植管理，强化质量意识，推广标准化种植，从而提升金柚品质，打响梅县金柚招牌，带动家乡人民致富。卢永昌说，自己也将一如既往关心、支持家乡的发展，多为家乡建设出力。

带团队创新求实 完成"不可能完成的任务"

在港珠澳大桥的建设中，卢永昌带领着中交四航院的技术团队主要负责两个人工岛和沉管隧道的建设。

这是世界级难度的工程，施工过程中遇到的许多难题也都是史无前例的。

首先是外在条件带来的难题，在外海建两个人工岛，任何国家都没有先例，一切都要靠团队自己摸索；海域地质条件复杂，软土分布范围广，软土层厚；东人工岛还处于中华白海豚核心保护区，环保要求非常高；建设海域位于珠江口，是我国乃至世上方预留出30万吨级的航道，必须创造性地建设世界上第一条深埋沉管隧道，同时也是界上交通最繁忙最复杂的水域之一。

还有技术上的难题。一般来说，沉管隧道的建设都是为了降低工程难度，沉管就铺设在海平面以下，埋的深度非常浅；而港珠澳大桥工程，由于要在沉管世界上最长的公路沉管隧道，沉管最深处达到了-50米，平均深度也达到了-28米。

在深槽铺设过程中，也发现了许多工程计划时没有预料到的问题，比如沉管下会有大量的淤积、珠江口也会不时出现2到3米的异常波浪，对施工的影响都很大。

钢圆筒快速沉岛多项方案世界首创

“在外人看来这几乎是不可能完成的任务，这次我们也完成了!”这一个个首次和首创，彰显的不仅仅是卢永昌团队的成绩，更是不屈不挠的工匠精神。

他带领设计团队数年奋战，在无前车之鉴，无经验积累的不利条件下，依靠自身力量攻坚克难。

突破了港珠澳珠大桥快速成岛多项技术瓶颈，多项技术世界首创；攻克了大超载比深井降水联合堆载预压加固人工岛及多种组合复合地基加固沉管隧道软土地基的技术难题。

“向前是远方，身后是故乡”

要说春节档电影《四海》最吸引广东人的

应该就是那片湛蓝的大海吧

这是一个一出门，四面都是海的地方

广东版的夏威夷原来在这里

春节档的电影大家看了多少部呢？每一部都各有各的优点，其中由刘昊然、刘浩存主演的《四海》话题度一直都很高。

早在大年初一上映前就已有近40W+的网友标注了“想看”。虽然口碑各有不同，但故事情节还是让文青和机车党们再一次的燃了起来。

拍摄地一贯延续了“故乡”“寻根”的感觉，延续着一贯的‘韩式’风格...能引起观众的无限遐想与深思。

最让人眼前一亮的是《四海》的拍摄地，便是位于北回归线上的南澳岛，用韩寒的形容则是“一出门哪儿都是海，就很‘四海'”。

南澳岛那湛蓝的天空与无边的海洋相连，岛屿上保留着独特的民俗风貌与安宁恬静的生活方式，岛民热情质朴。

这里作为刘昊然饰演的“阿耀”成长的地方，《四海》的故事发展也从这个美丽的小岛上开始，小岛的美丽景色让无数的观众为之向往。

南澳岛，是广东唯一的海岛县，处于闽、粤、台三地交界海面，自古便被称作“闽粤咽喉，潮汕屏障”，是名副其实的“南海绿洲”。

在《四海》的镜头下还有近期刘昊然MV拍摄的镜头下，这座隐世海岛的美貌得以呈现在更多人的眼前。

说起南澳岛，还记得有游客在岛屿上拍到了#5只粉色小海豚#，5只被誉为“水上大熊猫”的成年中华白海豚，成群结队地欢乐戏水，真是何其幸运。

一番美好的景象让人更加向往这个充满梦幻与浪漫的小岛。

/S336省道/

粤东最美环岛公路

来到南澳岛的都要尝试一次环岛自驾，环岛公路一边是山，另外一边则是无边的蔚蓝色的大海。

环岛路正是刘昊然在剧中飙车的拍摄点，虽然实际上这条公路不能飙摩托车，但也是自驾的好选择。

路上上贴心地设置了许多可供停车的观景平台，每个观景台都有 不一样的景色，给人一站一景的感觉。

/南澳大桥/

像巨龙一样的跨海大桥

南澳岛还拥有广东最大的跨海大桥——南澳大桥。从高处俯瞰这座桥，就像是一条长龙盘踞在大海之上。

像极了宫崎骏千与千寻中的海上列车，行驶在南澳大桥上，左右都能看到大海，无边无际，唯美又浪漫

/自然之门/

“立竿不见影”

每年夏至正午，当太阳直射北回归线时，日影将穿过“自然之门”上方圆球中心圆管，达到“立竿不见影”的奇迹。

/绝美灯塔/

无法忽视的存在

《四海》中，不经意就能看到灯塔的身影。通红的塔身，让人无法忽视它的存在。

在南澳岛上有不止一个灯塔，每个灯塔都有其特色，在灯塔上能感受到被大海包围的感觉。

潮汕地区给人的印象就是好多美食，南澳岛也不例外。南澳岛的海鲜，肥美又鲜嫩，且性价比也高。

// 蚝烙. //

从《舌尖上的中国》出圈的蚝烙是广东潮汕地区的传统小吃，鲜美酥香，酥而不硬，脆而不软。

// 南澳岛海鲜. //

来到南澳岛不能少了蒸石斑鱼、白灼大虾、蒸生蚝、炒葱姜花蟹、椒盐皮皮虾、生腌海鲜...

// 牛肉火锅. //

来到潮汕地区，没有吃到牛肉火锅可以说白来一趟了，潮汕人爱吃牛，更懂吃牛！

// 粿. //

在外地人看来，潮汕万物即可包进粿里，而且做法不同风味就不同，土豆粿、菜粿、麦粿、甜粿...可以吃个三天三夜吧。

刚过完春节，不少友友们都开始计划下一次旅行了吧，在三四月份来南澳岛就再适合不过啦。

这个“东方夏威夷”等你来解锁！

南澳岛，是广东唯一的海岛县，处于闽、粤、台三地交界海面，自古便被称作“闽粤咽喉，潮汕屏障”，是名副其实的“南海绿洲”。

在《四海》的镜头下还有近期刘昊然MV拍摄的镜头下，这座隐世海岛的美貌得以呈现在更多人的眼前。

近日，惠东县巽寮湾三角洲岛附近海域，被拍摄到五六只中华白海豚在海中游弋嬉戏。从拍摄画面中可以看到，白海豚在水中忽上忽下，不断跳跃，活力十足。

白海豚几乎每年都会在巽寮湾出现，每次都有五六只，这也证明了巽寮的海洋生态环境优越。

中华白海豚是唯一以“中华”名字命名的海洋豚类，也是中国海洋鲸豚中唯一的国家一级保护动物，是一种沿岸定居性的小型齿鲸类，属海洋哺乳动物，是世界上80余种鲸类之一，因其珍稀被称为“海上大熊猫”、“海上国宝”，沿海渔民以海上女神妈祖婆的名字，誉称它为“妈祖鱼”。

中华白海豚主要分布于西太平洋和印度洋的沿岸水域，属暖水性种类。我国的中华白海豚主要栖息在福建的厦门湾、东山湾、台湾岛的西部海域、广东的韩江口、珠江口、漠阳江口、雷州半岛东部海域、海南三亚附近海域以及广西北部湾等。目前，广东省中华白海豚数量超过2000头，是全国分布最密集的区域。

中华白海豚的身体似纺锤状，吻突狭长而侧扁，眼小呈椭圆形，外耳位于眼的后下方，呼吸孔呈新月形裂隙，背鳍略呈三角形，鳍肢短而宽，外缘弧形，内缘凸出。中华白海豚的皮肤会随着年龄增长而逐渐变成粉红色，因此又被人们称为“粉红海豚”。

中华白海豚不集成大群，常3~5只在一起，或者单独活动，偶见二十几只的大群，群体成员相对固定。由于生活水域中的猎物较为分散，中华白海豚经常要分头觅食。

龙骨帆船

"Keel Sailboat"

帆船运动是一项风、水、人、船四者完美结合，充满活力的运动。

帆船不同于平常的船只，有着高高的桅杆，会顺着风向缓缓地前行。船边溅起雪白晶莹的水花，海风吹鼓帆布，游弋于蔚蓝的大海，乘风破浪。

体验帆船，提前感受2022年亚青赛比赛项目，享受扬帆碧海的浪漫与激情。

问

帆船安全吗，会不会摇晃的很厉害？

答

安全。龙骨帆船船体很稳定，船长也很专业，体验过程中受风向和海浪影响会有一定的刺激，但很好玩，也不容易沾水湿身。

海上飞人

"Sea Flyer"

“海上飞人”是单人操作式喷射悬浮飞行器，利用水柱的喷射力作为动力，将人推向空中，让人体验如海豚跃出水面又潜入水中的感觉，也可以在数米高的空中做急转弯、盘旋、翻滚、滑行等灵活的精彩动作。

问

不会游泳可以体验海上飞人吗？

答

可以。此项目可以由教练带领游玩，无须担心是否会水。当然，勇于挑战的小伙伴也可以尝试自己体验，难度系数并不大哦。

摩托艇

"Motorboat"

摩托艇是集高科技、观赏、竞争和惊险刺激于一体的高速水上运动。

加速时摩托艇就如同离弦之箭在水面上飙浪，溅起的浪花不断拍打在身上，视线穿越层层水雾注视前方，感受马达轰鸣、风驰电骋的动感刺激！

问

摩托艇可以让游客自己体验驾驶吗？

答

可以。所有海上项目都有专业教练带领体验。可以在学习摩托艇驾驶技巧后，并在教练陪伴下，由游客亲手尝试开艇哦。

海上飞伞

"Sea Parachute"

“海上飞伞”由高速快艇在海面上牵引加速，迎风升上空中，体验像鸟一般在海天之上飞行，俯瞰山川大海，赋予全新的想象，这是一场刺激又安全的冒险之旅！

水上沙发、香蕉船

"Floating Sofa"

想象一下，在海上坐沙发是一种什么样的感觉？在摩托艇的牵引下，惬意地躺在有靠背、有扶手的酷炫“大黄鸭”沙发上，或是形似香蕉的橡胶皮筏上，在海上肆意地疾驰，迎着清新的海风和起伏的波浪，既畅快又刺激！

沙滩公园有休息和吃东西的地方吗？

公园岸堤上的海岸音乐吧，提供奶茶、小食、南澳特色饮品等；还有太阳伞、沙滩椅、功夫茶等服务，吹着海风，听着海浪，感受心旷神怡的度假时光~

牵引滑水、桨板

"Drawing Water Skiing"

滑水运动是人借助动力的牵引在水面上“行走”的水上运动，需要一定的平衡力和技巧。手握住缰绳，脚部和滑水板绑定，在摩托艇的牵引下，开啔刺激的海上冲浪之旅。

桨板源于美国夏威夷，近年逐渐风靡全球。牵引桨板比起滑水更容易尝试，更适合作为新手小白的冲浪初体验。

问

体验滑水冲浪有什么要求？

答

进阶水上运动项目，建议体验者年龄为16-40周岁之间，建议体重小于80公斤。有心脏病、高血压、癫痫、严重关节炎等疾病患者不可体验。

豪华快艇

潮汕文化，隶属于岭南文化，广东三大文化之一。是汉文化的子文化，是古中原文化的遗存，是中华民族优秀文化的重要组成部分，在历代传承过程中不断发展而形成的汉文化。潮汕文化是潮汕人(潮汕民系)创造的文化，有中外文化兼容的特点， 特征是以海洋文化为主。潮汕文化发源并流行于广东省潮汕地区(汕头、潮州、揭阳、汕尾) 和梅州市的丰顺县汤南、留隍两镇 、以及惠州市的惠东县部分地区。潮汕文化既具有鲜明的地域特色，又带有中原文化的某些特点，贯穿于潮人整个社会生活之中。以潮汕英歌舞、工夫茶、潮汕饮食、潮汕商帮文化、潮汕善堂、潮汕祠堂，潮汕方言、潮绣、潮剧、潮乐、潮汕工艺、潮汕木雕、潮州大锣鼓、潮汕民居等为代表的潮汕文化影响深远，誉播海内外。香港潮人盂兰胜会 、 潮汕英歌舞、潮汕侨批文化 等多项被列为国家非物质文化遗产 。潮汕文化遗产是中华的瑰宝。截至目前，粤东四市列入国家级的非遗名录已达46项，列入省级名录的达132项。与此同时，有29人被国家文化部命名为国家级非物质文化遗产项目代表性传承人。1907年5月22日，丁未黄冈起义爆发，这是孙中山先生亲自领导的推翻清王朝的武装起义，乃日后诸多民主武装革命的先声。但《南洋总汇报》却大肆指责孙中山勾结"潮汕会党"，策动黄冈起义。"潮汕"之名由此见诸报端，广为人知。

近年来，在汕头南澳海域，海豚的“出镜率”非常高，很多人对这群海洋精灵都不陌生了不过，“大豚背小豚”的景象，见过的人并不多。3月5日，有街坊就拍到了这神奇的一幕，而这背后，还有令人意外的原因。

3月5日下午，在汕头南澳大船澳附近海域上出现了一群海豚，令人惊奇的是，其中一只成年海豚，还背着一只“幼豚”，只见成年海豚不断地将这只“小豚”托出水面，似乎是在尝试施救。

据介绍，中华白海豚是一个近岸物种，常常出现在汕头南澳大桥一带，而江豚则生活在深一点的海域，比如南澎列岛周边。专家认为，这一次可能是江豚误打误撞来到了近岸，因此受了伤。

南澎列岛海洋生态国家级自然保护区管理局 张喆大：因为江豚是大洋性的生活区域，进入近岸了它有些不适应，不适应近岸的咸度还有生活习性，包括那些礁石和波浪，它会影响江豚的判断，很容易就误撞，或者被渔具等什么东西给伤到，容易导致死亡。

海豚，自古被人称为“义鱼”，它们托举溺水人类或是同类幼崽的事迹时有发生。专家指出，这是海豚的一种本能。

南澎列岛海洋生态国家级自然保护区管理局 张喆大：像中华白海豚的智力，它是达到了人类四五岁、五六岁小孩的智商水平，我们叫做“妈祖鱼”。是因为它有这个救护的想法，尽量会把它们救护到安全的地方，曾经它也有救过人类，或者救过陆生动物，好多视频都有，海豚救助狗、人类、其他的都有。
